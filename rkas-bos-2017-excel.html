6. jelaskan perbedaan dari tampilan microsoft excel 2017 dengan microsoft excel 2010


Microsoft Excel 2010
a. Tahun: 2010.
b. Versi Excel: Excel 2010 (Excel 14).
c. Sistem Operasi: Windows 7, Windows Vista(dengan Service Pack 2), Microsoft Windows XP, Windows Server 2008, Windows Server 2008 R2, dan Windows 8.
d. Versi Microsoft Office: Microsoft Office 2010.

Microsoft Excel 2016
a. Tahun: 2016
b. Versi Excel: Excel 2016 (Excel 16)
c. Sistem Operasi: Windows 7Windows Server 2008 R2, Windows Server 2012, dan Windows 8.
d. Versi Microsoft Office: Microsoft Office 2016 & Office 365.

⚠ Belum ada versi excel 2017.

Fitur dan sorotan Excel 2010
1. Perubahan antarmuka pengguna

Pita, diperkenalkan di Office 2007, dapat dipersonalisasi di Excel 2010. Buat tab dan grup ubahsuaian dan ubah urutan atau ganti nama tab built-in. Tab File ditambahkan untuk membuka tampilan Office Backstage, di mana Anda dapat membuat dan membuka file, mencetak, mengatur opsi Excel, dan banyak lagi. Fitur dokumen yang terpercaya memudahkan untuk membuka buku kerja dan dokumen lainnya yang berisi konten aktif. Dan tampilan yang terlindungi memungkinkan Anda membuat keputusan yang lebih tepat tentang membuka file.

2. Fitur analisis data interaktif

Fitur seperti slicers dan sparklines ditambahkan untuk membantu Anda memfilter data secara intuitif dan meringkas tren secara visual. Format kondisional yang disempurnakan dan set ikon baru mempermudah penarikan rentang data yang menarik. PivotTables jauh lebih mudah digunakan dan lebih responsif daripada versi sebelumnya-dengan pengambilan data lebih cepat dan penyaringan yang disempurnakan.

3. Fitur dan fungsi lembar kerja

Excel 2010 menyertakan fungsi statistik dan lainnya yang lebih akurat. Beberapa fungsi yang ada diganti namanya untuk lebih menggambarkan apa yang mereka lakukan.Alat pengeditan persamaan berguna untuk memasukkan persamaan matematis ke dalam lembar kerja atau membuat persamaan Anda sendiri dengan menggunakan perpustakaan simbol matematika. Peningkatan efek artistik dan alat editing gambar membantu Anda mengkomunikasikan gagasan Anda menggunakan foto, gambar, dan SmartArt.

Fitur dan sorotan Excel 2016
1. Semua pemahaman berada dalam data

Atur tata letak data Anda

Atur data numerik atau teks di lembar bentang atau buku kerja. Menampilkan data dalam konteks akan membantu Anda membuat keputusan berdasarkan informasi.

Format dan susun ulang data

Saat Anda melihat konfigurasi yang berbeda, Excel mempelajari dan mengenali pola Anda dan melengkapi otomatis data sisanya untuk Anda. Tidak ada rumus atau makro yang diperlukan. Fitur pencarian Beri Tahu Saya memandu Anda pada perintah fitur yang Anda perlukan untuk mendapatkan hasil yang Anda cari.

Lakukan analisis

Excel akan menjalankan analisis kompleks dan meringkas data dengan pratinjau berbagai pilihan pivot-table, sehingga Anda dapat membandingkannya dan menentukan salah satu pilihan yang paling sesuai dengan kebutuhan.

2. Dapatkan gambaran yang lebih baik dari data Anda

Gambarkan data dalam diagram dan grafik

Excel dapat menyarankan diagram dan grafik yang paling menggambarkan pola data Anda. Lihat pratinjau pilihan dengan cepat, lalu pilih salah satu yang paling menggambarkan informasi Anda dengan jelas.

Temukan alur cerita terbaik Anda

Temukan dan bandingkan berbagai cara untuk menggambarkan data dan ide secara visual. Ketika melihat salah satu yang memperlihatkan data terbaik, terapkan pemformatan, grafik mini, bagan, dan tabel dalam sekali klik. Serangkaian bagan dan grafik modern baru di Excel 2016 membantu Anda menyajikan data dalam cara terbaru. ￼

Soroti tren dan pola

Permudah proses pemahaman tren dan pola dalam data Anda dengan bilah, warna, dan ikon untuk menyoroti nilai penting secara visual. Fitur prakiraan sekali klik baru di Excel 2016 menciptakan prakiraan di seri data Anda dengan sekali klik untuk tren mendatang.

3. Tambahkan orang lain untuk meninjau

Berbagi dari awan

Pastikan setiap orang memiliki versi buku kerja terbaru dengan membagikannya di awan menggunakan OneDrive atau SharePoint, sehingga orang lain dapat menampilkan, mengedit, dan bekerja sama. ￼ Atau cukup kirimkan email maupun pesan instan sebagai lampiran.

Bekerja sama secara real-time

Setelah lembar bentang disimpan ke OneDrive, OneDrive for Business, atau SharePoint, Anda dan tim dapat bekerja bersama secara real-time dengan Excel Online. Saat melakukan pengeditan dan perubahan di dokumen, riwayat versi di Excel 2016 yang telah disempurnakan memungkinkan Anda melihat atau kembali ke draf sebelumnya. ￼

Jawaban:

Data Teks (Label), Data Angka (Value), Data waktu (time) dan Data Tanggal (Date), dan Tipe Data logika

Penjelasan:

1. Tipe Data Text (Label)

Jenis data label sering juga disebut sebagai data Character, String atau Teks.

Tipe Data Label pada excel adalah jenis data yang tersusun oleh kumpulan karakter huruf (A-Z/a-z), angka(0-9) atau karakter simbol(!,@,#, dll.).

2. Tipe data angka (value)

Tipe data value disebut juga sebagai data numerik atau angka yakni data pada sel excel yang karakter penyusun utamanya berupa nilai angka 0-9.

3. Tipe Data tanggal & waktu (Date & Time)

Jika anda memasukkan data yang persis atau mirip dengan format tanggal dan atau waktu ke dalam sel lembar kerja excel, maka mcirosoft excel akan secara otomatis menkonversi input data tersebut ke dalam data value tipe tanggal dan atau waktu.

contoh: 01/01/2015, 1 januari 2015, 12:00, 14:30:00, 01/01/2015 12:00, dsb.

4. Tipe Data Logika

Data tipe ini sering disebut juga sebagai data Boolean. Jika anda mengetikkan teks TRUE atau False ke dalam sel maka excel akan menganggap teks tersebut sebagai tipe numeric dimana TRUE bernilai angka 1 dan FALSE bernilai angka 0.
