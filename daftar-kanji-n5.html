4. jlpt N5 perlu kanji atau tidak/kalau perlu apa aja? Maaf kalau penulisan/kalimat salah


Untuk JLPT N5, yang jelas kalian harus hafal Kana (Hiragana & Katakana). Selain itu, kalian juga harus menghafal beberapa kanji dasar. Kanji-kanji di level N5 tingkat kesulitannya termasuk mudah. Jumlah coretan kanji nya tidak sebanyak level N4 ke atas.

Kanji di level N5 jumlahnya ada sekitar 100an. Masing-masing kanji nya juga memiliki beragam cara baca dan makna. Contoh-contoh kanji level N5 antara lain sebagai berikut.

日 (Nichi / Hi) = Hari, Matahari,...一 (Ichi / Hito) = Satu二 (Ni / Futa) = Dua三 (San / Mi) = Tiga山 (Yama / San) = Gunung川 (Kawa / Sen) = Sungai食 (Shoku / Ta) = Makandll...

Selain menghafal kanji, kalian juga perlu menghafal beberapa tata bahasa, dll. Di level N5 ini adalah kumpulan materi dasar dalam Bahasa Jepang, jadi nggak akan sesusah level lainnya.

Jawaban & Penjelasan  

    Kanji merupakan salah satu jenis huruf Jepang, berupa karakter pinjaman dari Cina. Masing-masing huruf kanji memiliki beragam cara baca dan makna. Cara baca kanji dibagi menjadi Kunyomi (cara baca Jepang) dan Onyomi (cara baca Cina). Jumlah kanji yang biasa digunakan ada sekitar 2000 kanji. Kanji dibagi menjadi 5 level, dari yang paling rendah N5, sampai yang paling tinggi N1.

Berikut 10 contoh kanji level N5 :

========

1). 人

Arti = Orang

Kunyomi = Hito

Onyomi = Jin, Nin

   Contoh :

人間 (Ningen) = Manusia人生 (Jinsei) = Kehidupan

========

2). 山

Arti = Gunung

Kunyomi = Yama

Onyomi = San

   Contoh :

富士山 (Fujisan) = Gunung Fuji山田 (Yamada) = Yamada *nama

========

3). 川

Arti = Sungai

Kunyomi = Kawa

Onyomi = Sen

   Contoh :

川魚 (Kawazakana) = Ikan sungai鴨川 (Kamogawa) = Sungai Kamo

========

4). 生

Arti = Lahir, hidup, tumbuh

Kunyomi = I, U, Ha

Onyomi = Sei, Shou

   Contoh :

先生 (Sensei) = Pak/Bu (guru, dokter, dll)年生 (Nensei) = Kelas *tingkatan

========  

5). 日

Arti = Hari, matahari, minggu, Jepang

Kunyomi = Hi, Bi, Ka

Onyomi = Nichi, Jitsu

   Contoh :

日曜日 (Nichiyoubi) = Hari Minggu今日 (Kyou) = Hari ini

========

6). 月

Arti = Bulan, senin

Kunyomi = Tsuki

Onyomi = Getsu, Gatsu

   Contoh :

四月 (Shigatsu) = Bulan April今月 (Kongetsu) = Bulan ini

========

7). 火

Arti = Api, selasa

Kunyomi = Hi, Bi, Ho

Onyomi = Ka

   Contoh :

火曜日 (Kayoubi) = Hari Selasa花火 (Hanabi) = Kembang api

========

8). 水

Arti = Air, rabu

Kunyomi = Mizu

Onyomi = Sui

   Contoh :

水曜日 (Suiyoubi) = Hari Rabu鼻水 (Hanamizu) = Ingus

========

9). 土

Arti = Tanah, sabtu

Kunyomi = Tsuchi

Onyomi = To, Do

   Contoh :

土曜日 (Doyoubi) = Hari Sabtu土地 (Tochi) = Tanah

========

10). 花

Arti = Bunga

Kunyomi = Hana

Onyomi = Ka, Ke

   Contoh :

花火 (Hanabi) = Kembang api花びら (Hanabira) = Kelopak bunga

========

Semoga membantu.. ^^
