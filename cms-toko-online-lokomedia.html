11. Salah satu cms yang di gunakan untuk kelas online


Jawaban:

Moodle

Penjelasan:

Moodle adalah sebuah Course Open Source Management System (CMS), juga dikenal sebagai Learning Management System (LMS) atau Virtual Learning Environment (VLE). Hal ini telah menjadi sangat populer di kalangan pendidik di seluruh dunia sebagai alat untuk membuat situs web online yang dinamis bagi siswa mereka. Untuk bekerja, perlu diinstal pada web server di suatu tempat, baik di salah satu komputer Anda sendiri atau salah satu di sebuah perusahaan web hosting.

Fokus dari proyek Moodle selalu untuk memberikan pendidik alat terbaik untuk mengelola dan mempromosikan belajar, tetapi ada banyak cara untuk menggunakan Moodle:

Moodle memiliki fitur yang memungkinkan untuk skala penyebaran yang sangat besar dan ratusan ribu mahasiswa, namun juga dapat digunakan untuk sekolah dasar atau hobi pendidikan.

Banyak institusi menggunakannya sebagai platform mereka untuk melakukan kursus sepenuhnya online (termasuk di sini adalah kampus admin), sementara beberapa menggunakannya hanya untuk menambah face-to-face program (dikenal sebagai blended learning).

Banyak pengguna kami suka menggunakan modul kegiatan (seperti forum, database dan wiki) untuk membangun komunitas kolaboratif kaya belajar sekitar materi pelajaran (dalam tradisi konstruksionis sosial), sementara yang lain lebih suka menggunakan Moodle sebagai cara untuk menyampaikan konten kepada siswa (seperti paket SCORM standar) dan menilai pembelajaran menggunakan tugas atau kuis.
