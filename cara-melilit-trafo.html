1. Arus Primer sebuah trafo adalah 1A sedangkan arus Sekundernya adalah 0,5A. Jika lilitan di Primer Trafo tersebut adalah 200 lilitan dan 500 lilitan dikumparan sekundernya. Berapakah Efisiensi Trafo tersebut? (berikan cara)​


Jawaban:

Penjelasan:

Diketahui :
Arus primer (Ip) = 1 A
Arus sekunder (Is) = 0,5 A
Lilitan primer (Np) = 200 lilitan
Lilitan sekunder (Ns) = 500 lilitan

Ditanya: Effisiensi trafo (ɳ) ?

Jawab :
ɳ = (Ns x Is / Np x Ip) x 100%
ɳ = (500 x 0,5 / 200 x 1) x 100%
ɳ = (250 / 200) x 100%
ɳ = 1,25 x 100%
ɳ = 125 %

Efisiensi Trafo tersebut adalah 125%
