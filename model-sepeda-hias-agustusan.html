2. Banyak sepeda model B yang dapat dibuat dengan modal tersebut adalah


x=banyakynya sepeda model A
y=banyaknya sepeda model B

Didapat 2persamaan linier dan gunakan metode eliminasi
x+y=25 =====================>x3.000.000
3.000.000x+4.000.000y=96.000.000 =>×1
3.000.000x+3.000.000y=75.000.000
3.000.000x+5.000.000y=96.000.000 _
-1.000.000y=-21.000.000
y=21

jadi banyaknya sepeda model B=21Persamaan Linear 2 Variabel

Misal
Model A= X
Model B= Y

X≤1 1/2 B
X+Y≤25
3.000.000X+4.000.000Y≤96.000.000
3X+4Y=96 ( dalam jutaan )
Model B=....

X+Y=25
X=25-Y
3(25-Y)+4Y=96
75-3Y+4Y=96
-3Y+4Y=96-75
Y=21

X=25-Y
X=25-21
X=4

Jadi Sepeda Model B adalah 21 unit dan Sepeda Model A adalah 4 unit.
Pilihan B.
