Pilih “I Accepted the terms of the license agreement” dan klik “Continue”Klik OK untuk memulai memformat USB Flashdisk dan file2 windows 7 secara otomatis akan dimasukkan ke USB Flashdisk.

Cara Manual:
1. Colokkan USB Flaskdisk anda. (masukin ke usb port yah, jgn ke tempat lain).
2.Tekan Tombol WIN+R, ketik cmd dan klik OK.
3.Ketik diskpart dan tekan ENTER.
4.Ketik list disk, tekan ENTER dan pilihlah yang mana adalah USB flashdisk anda, hati2 jangan salah pilih nanti salah2 yg keformat adalah disk yang lain. jikalau anda punya hanya 1 harddisk makan USB flaskdisknya adalah disk1.
5,Ketik select disk 1 dan tekan ENTER.
6.Ketik clean dan tekan ENTER.
7.Ketik create partition primary lalu tekan ENTER.
8.Ketik select partition 1 dan tekan ENTER
9.Ketik active dan tekan ENTER.
10.Ketik format fs=fat32 dan tekan ENTER
11.Ketik assign dan tekan ENTER
12.Ketik exit dan tekan ENTER.
13.Masukkan disk DVD windows 7  dan copy lah semua isi DVD tadi ke USB Flaskdisk anda.
14.Booting komputer anda melalu USB Flashdisk, setting malalui bios komputer anda, pastikan boot nya melalui USB Flashdisk, apabila langkah2nya betul, maka proses instalasi windows seharusnya berjalan melalui USB Flashdisk
Cara Otomatis.
Cara otomatis tentunya merupakan pilihan saya, dan untuk itu sudah tersedia program yang namanya WinToFlash, program ini akan melalukan langkah2 seperti cara manual diatas, yang kita perlu lakukan hanyalah menjalankan programnya, pilih sumber windows 7 dan arah USB flashdisk.
1. Unduh dulu program WinToFlash versi terbaru. (download link diakhir artikel)
2.Ekstrak dan jalankan file WinToFlash.exe
3.Klik tombol “Check” dan mulailah proses Windows setup transfer wizard.
4,Klik “Next”
5.Pilih lokasi file Windows 7 dan lokasi USB Flaskdisk anda. klik “Next”.
6.Pilih “I Accepted the terms of the license agreement” dan klik “Continue”
7.Klik OK untuk memulai memformat USB Flashdisk dan file2 windows 7 secara otomatis akan dimasukkan ke USB Flashdisk.
8.Klik “Next” apabila selesai pengopian, dan bootinglah komputer anda melalui USB Flashdisk tadi. apabila langkah2nya betul, maka proses instalasi windows seharusnya berjalan melalui USB Flashdisk.

15. Untuk menginstal OS dengan menggunakan flashdisk apa yang harus dilakukanagar boot pertama adalah flashdisk​


Penjelasan:

1. Siapkan file ISO Windows 10.

2. Satu hal yang sangat penting jika kitaingin menginstal windows 10 dengan lampu kilat berkedip sendiri Gunakan flash dengan kondisi masih baik dan dengan kapasitas setidaknya 8GB.

3. Ambil Rufus. Rufus adalah aplikasi yang digunakan untuk membuat flash bootable. Ada banyak aplikasi yang dapat mengubah kita ke flash flashdisk bootable. Namun, Rufus dipilih oleh pengguna sebagai salah satu yang terbaik aplikasi gratis

4. Setelah men-download rufus, maka langkah selanjutnya adalah pasang tongkat ke port USB. Kemudian buka rufus aplikasi.5. Masukkan file ISO Windows 10 yang telah Anda download.

6. Ikuti konfigurasi di bawah ini dan kemudian klik START7. Setelah melakukan langkah-langkah sebelumnya, kita tinggal menunggu sampai proses selesai.

7. Setelah melakukan langkah-langkah sebelumnya, kita tinggal menunggu sampai proses selesai.8. Jika proses telah selesai dan semua langkah telah dilakukan dengan benar, maka flashdisk dapat digunakan untuk menginstal Windows 10 pada pc/laptop kita.

Setelah berubah menjadi flash flashdisk bootable kita, maka kita segera dapat memulai proses menginstal Windows 10 dengan flash melalui BIOS. Tahap pertama yang harus dilakukan adalah untuk mengatur BIOS untuk boot dengan flashdisk. Langkah yang harus dilakukan pada tahap ini adalah:

1. Pastikan flash drive bootable terpasang port USB laptop, dan kemudian restart laptop.

2. Untuk memasuki BIOS, tekan DEL (Delete). Selain tombol Del, ada tombol lain yang dapat digunakan, yaitu F1, F2, atau F10, tergantung pada jenis laptop yang ada.

3. Setelah menampilkan layar BIOS, kemudian pindah ke tab “Boot” menggunakan tombol ◄ ►

4. Setelah memilih tab Boot, maka akan muncul menu Boot. boot pertama akan dilakukan pada hard drive.

5. Tempatkan “Removable Devices” ke atas dengan memilih Removable Devices dan bergeser dengan tombol +/-, untuk boot pertama kali dilakukan di flashdisk. BIOS dapat mendeteksi segera flashdisk jenis / merek.

6. Jika langkah 5 telah dilakukan, maka kita Bisa langsung menyimpan pengaturan dengan memilih tab Exit dan pilih Exit Saving Changes dan pilih Yes. Atau jika Anda ingin lebih praktis, kita dapat langsung menekan tombol F10 dan pilih opsi Ya.
