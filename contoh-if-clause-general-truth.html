7. 10 contoh percakapan if clause reminder, an imperative, suggestion dan general truth dan dream


IF CLAUSE AS REMINDER, IMPERATIVE, SUGGESTION AND GENERAL TRUTH

 

Jawaban  Pendahuluan  

If clause atau Conditional sentence selain digunakan sebagai sebuah perumpamaan juga bisa digunakan sebagai pengingat (reminder), saran (suggestion),  perintah (imperative), impian (dream) dan fakta (general truth).

Pembahasan  

If Clause sebagai Reminder atau Pengingat

If clause sebagai reminder digunakan untuk mengingatkan seseorang terhadap sesuatu dengan tujuan agar mendapatkan hasil yang terbaik.

Rumus dalam If clause as reminder adalah:

If + Simple Present, Subject + have/has to atau must + V1

Contoh:

1. If you want to get the highest score, you must study harder.

2. If you want to be a champion, you have to train hard.

3. If she wants to visit me, she must come to my house around 4 pm.


If Clause sebagai Imperative atau Perintah

If clause yang digunakan sebagai perintah bertujuan agar seseorang mematuhi suatu peraturan atau perintah. Kalau orang tersebut mengikuti perintahnya, maka akan hasil yang baik untuknya.

Rumus untuk If clause as imperative:

If + Simple present, V1 + Object / Adverb (imperative)

Contoh:

1. If you want to pass the University enrollment test, stop playing with your mobile phone!

2. If you want to be healthy, eat some nutritious foods and fruits!

3. If you want to buy a car, start saving your money!


If Clause sebagai Suggestion atau Saran

If Clause sebagai suggestion digunakan untuk memberi saran agar seseorang mendapatkan apa yang dia inginkan.

Rumus untuk If clause as Suggestion:

If + Simple Present, S + should/should not + V1

Contoh:

1. If you want to get the highest score, you should study harder.

2. If you want to speak English fluently, you should do more conversation with native English.

3. If you want to be healthy. you should eat more fruits.


If Clause sebagai General Truth atau Fakta

If clause sebagai general truth sering disebut juga sebagai conditional tipe zero. Ungkapan ini menyatakan sebuah fakta. Biasanya hubungan sebab-akibat berlaku dalam tipe ini.

Rumus If clause as General Truth:

If + simple present, simple present

Contoh:

1. If you play with fire, you get burned.

2. If you heat the ice, it melts

3. If you mix yellow and blue, you get green.


If Clause sebagai Impian (Dream)

If clause sebagai dream atau impian digunakan untuk menyatakan sebuah impian dari seseorang di masa depan. If clause ini sering disebut conditional tipe satu.

Rumus If clause as Dream:

If simple present, simple future

Contohnya:

1. If I have much money, I will travel around the world.

2. If I marry that girl, I will be very happy.

3. If you can't come, They will be sad
