1. huruf - huruf jepang dari a-z


Huruf di jepang tidak terdiri dari a-z, melainkan hanya vokal berakhiran "a-i-u-e-o"
Sisanya, kita hanya perlu menambah huruf di depan nya, yaitu: "k-s-n-h-m-y-r-g-z-b-p" . jadi nanti akan berubah menjadi (contoh) "So / そ"
-Untuk deretan "H", hu itu tidak ada. Diganti menjadi "fu"
-di Jepang tidak ada hurum "L". Maka dari itu akan diganti menjadi " R"
-adapun huruf yang memiliki 3 vocal (hanya a-u-o) yaitu "Y- ky - sh - ch - ny - hy - my - ry - gy - jy - by - py"

❇Jenis Huruf Jepang
Di bahasa jepang terrbagi 4 jenis huruf, yaitu:
↪Hiragana: hruf jepang yang dipakai sehari-hari. Contoh: する/suru
↪Katakana: huruf yang digunakan jika ada bahasa serapan asing. Contoh: テレビ/terebi (tv)
↪Kanji: huruf dari china yang masuk ke Jepang untuk lebih mempersingkat hiragana. Contoh: にほん↔日本 / Nihon (Jepang)
↪romaji: bahasa latin. Contoh: Hikari (cahaya)

2. Sebutkan huruf jepang dari a-z


Huruf di Jepang tidak terdiri dari A-Z. Huruf mereka memiliki ciri khas tersendiri. 1 huruf mereka bisa sikatakan satu kata di huruf normal.

Tetapi, di Jepang memiliki 5 Jenis penulisan huruf.

=== Jawaban ===

❇ Kanji
↪Huruf ini adalah huruf serapan dari China yang di sebut Zanshi. Ada beberapa huruf Kanji (dari china) yang penyebutannya di pakai di Jepang, dan ada pula yang berubah penyebutan. Itu di sebut sebagai "Konyomi dan Onyomi"
Contoh kanji;
-白い約束 (Shiroi Yakusoku): janji putih / janji suci

Huruf kanji untuk saat ini sekitar 2.230 kanji dan dikelompakn menjadi:
N5 = 103 Kanji (dasar)
N4 = 181 Kanji
N3 = 324 Kanji
N2 = 415 Kanji
N1 = 1.207 Kanji

Jika jenis pembacaan kanji, dibagi menjadi 2. yaitu:
• Kunyomi
• Onyomi

❇ Hiragana
↪Huruf ini ialah huruf Jepang asli. Secara sejarah, Hiragana di buat oleh para wanita Jepang untuk mempermudah tulisan kanji. Terdapat 46 huruf dasar.
Contoh hurufnya:
- なないろ (Nanairo): 7 macam / 7 warna

❇ Katakana
↪Huruf ini ialah huruf untuk serap asing. Secara sejarah, huruf ini di buat oleh para lelaki Jepang, maka dari itu penulisna nya sedikit lebih tegas. Fungsi huruf ini ialah untuk menulis nama orang, negara, wilayah, wisata alam, rekreasi, produk luar Jepang yang semisalkan "diubah" ke Bahasa Jepang tetapi agak "memaksa". Terdapat 46 huruf dasar.

❇ Romaji
↪Roma di ambil dari kata latin, ji adalah huruf. Berarti mengartikan: huruf latin. Biasanya ini berfungsi bagi yang gak bisa baca hiragana, katakana, ataupun kanji
Contoh huruf:
↪Nagai Hikari: cahaya Panjang

❇ Furigana
↪Huruf ini sebenarnya terdiri dari hiragana. Tetapi, maksud dari furigana ini adalah "teks kecil". Di setiap buku Jepang, ada keterangan furigana di atas kanji dengan ukuran keci. Fungsinya agar seseorang yang tak bisa membaca kanji dapat membaca buku atau teks tersebut.

9. sebutkan huruf A sampai Z dalam bahasa jepang


Huruf Jepang tidak sama dengan huruf abcd yang biasa kita pakai, jadi tidak ada huruf A sampai Z dalam Bahasa Jepang. Lalu bagaimana huruf-huruf Jepang? Simak pembahasan berikut.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

PENDAHULUAN

Huruf yang digunakan di Jepang ada 3 jenis, yaitu Hiragana, Katakana, dan Kanji. Huruf Hiragana dan Katakana disebut juga Kana. Kana memiliki satu bunyi tiap hurufnya, dan tidak memiliki makna. Sedangkan Kanji bisa memiliki lebih dari satu bunyi tiap hurufnya, dan kebanyakan memiliki makna. Selain ketiga huruf tersebut, ada 1 jenis huruf lagi yang hanya digunakan orang luar Jepang. Huruf ini berbentuk abcd, dan dinamakan Romaji. Agar lebih jelas, berikut saya jelaskan lebih lanjut mengenai keempat huruf ini.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

PEMBAHASAN1.  ひらがな (Hiragana)

    Merupakan huruf asli Jepang yang digunakan untuk menulis kosakata umum, partikel, furigana, okurigana, dll. Garis-garis pada huruf Hiragana kebanyakan melengkung. Huruf Hiragana masing-masingnya hanya memiliki satu bunyi saja, dan tidak memiliki makna. Huruf-huruf Hiragana dibagi menjadi Gojuuon, Dakuon, Handakuon, Youon, Sokuon, dan Chouon. Yang akan saya sebutkan hanya Gojuuon saja, karena meruapakan huruf dasarnya, sebagai berikut.

あ い う え お (a - i - u - e - o)か き く け こ (ka - ki - ku - ke - ko)さ し す せ そ (sa - shi - su - se - so)た ち つ て と (ta - chi - tsu - te - to)な に ぬ ね の (na - ni - nu - ne - no)は ひ ふ へ ほ (ha - hi - fu - he - ho)ま み む め も (ma - mi - mu - me - mo)や ゆ よ (ya - yu - yo)ら り る れ ろ (ra - ri - ru - re - ro)わ を (wa - wo)ん (n)

-

2.  カタカナ (Katakana)

    Merupakan huruf asli Jepang yang digunakan untuk menulis kosakata serapan dari luar Jepang, seperti nama non-Jepang, nama negara, nama-nama ilmiah, dll. Selain itu juga digunakan untuk memberi penekanan pada suatu kata. Garis-garis pada huruf Katakana lebih lurus dan kaku. Huruf Katakana masing-masingnya juga hanya memiliki satu bunyi saja, dan tidak memiliki makna. Huruf-huruf Katakana juga dibagi menjadi Gojuuon, Dakuon, Handakuon, Youon, Sokuon, dan Chouon. Yang akan saya sebutkan hanya Gojuuon saja, karena meruapakan huruf dasarnya, sebagai berikut.

ア イ ウ エ オ (a - i - u - e - o)カ キ ク ケ コ (ka - ki - ku - ke - ko)サ シ ス セ ソ (sa - shi - su - se - so)タ チ ツ テ ト (ta - chi - tsu -te - to)ナ ニ ヌ ネ ノ (na - ni - nu - ne - no)ハ ヒ フ ヘ ホ (ha - hi - fu - he - ho)マ ミ ム メ モ (ma - mi - mu - me - mo)ヤ ユ ヨ (ya - yu - yo)ラ リ ル レ ロ (ra - ri - ru - re - ro)ワ ヲ (wa - wo)ン (n)

-

3.  漢字 (Kanji)

    Merupakan huruf yang diadopsi dari huruf Cina, yaitu Hanzi. Huruf Kanji digunakan untuk menulis hampir semua kosakata Bahasa Jepang. Huruf Kanji masing-masingnya bisa memiliki lebih dari satu bunyi, dan memiliki makna. Huruf Kanji juga kebanyakan lebih rumit daripada Kana, maka dari itu banyak orang yang malas belajar Kanji (termasuk saya :v). Cara baca Kanji dibagi menjadi Kunyomi dan Onyomi. Jumlah Kanji ada puluhan ribu, tapi yang biasa digunakan hanya sekitar 2000 Kanji. Beberapa contoh huruf Kanji tentang nama-nama hari, sebagai berikut.

月 (Getsu) = Senin火 (Ka) = Selasa水 (Sui) = Rabu木 (Moku) = Kamis金 (Kin) = Jum'at土 (Do) = Sabtu日 (Nichi) = Minggu

*Kanji-kanji tersebut masih memiliki bunyi dan makna lain, selain yang sudah saya sebutkan.

-

4.  ローマ字 (Romaji)

    Merupakan huruf yang digunakan untuk menulis cara baca dari huruf-huruf Jepang dalam huruf abcd. Huruf Romaji tidak biasa digunakan di Jepang, jadi hanya digunakan di luar Jepang saja. Orang-orang luar Jepang menggunakan huruf Romaji untuk mempermudah mereka dalam membaca huruf Jepang. Contoh dari huruf Romaji yaitu seperti yang terdapat dalam tanda kurung di sebelah huruf-huruf Jepang yang saya sudah sebutkan sebelumnya di atas.

Nah, ini yang akan saya jelaskan, Huruf Jepang tidak terdiri dari Huruf Alphabet, Di Huruf Bahasa Jepang memerlukan bantuan huruf vokal. Lantas apa itu Huruf Hiragana? Hiragana, adalah bentuk tulisan Jepang yang berfungsi sebagai untuk menulis sesuatu dari Jepang, seperti nama, menulis benda, dan lain sebagainya.

14. Huruf Alfabet dari A sampai Z dalam bahasa Jepang.


Huruf di Jepang tidak terdiri dari A-Z. Huruf mereka memiliki ciri khas tersendiri. 1 huruf mereka bisa sikatakan satu kata di huruf normal.

Tetapi, di Jepang memiliki 5 Jenis penulisan huruf.

=== Jawaban ===

❇ Kanji
↪Huruf ini adalah huruf serapan dari China yang di sebut Zanshi. Ada beberapa huruf Kanji (dari china) yang penyebutannya di pakai di Jepang, dan ada pula yang berubah penyebutan. Itu di sebut sebagai "Konyomi dan Onyomi"
Contoh kanji;
-白い約束 (Shiroi Yakusoku): janji putih / janji suci

Huruf kanji untuk saat ini sekitar 2.230 kanji dan dikelompakn menjadi:
N5 = 103 Kanji (dasar)
N4 = 181 Kanji
N3 = 324 Kanji
N2 = 415 Kanji
N1 = 1.207 Kanji

Jika jenis pembacaan kanji, dibagi menjadi 2. yaitu:
• Kunyomi
• Onyomi

❇ Hiragana
↪Huruf ini ialah huruf Jepang asli. Secara sejarah, Hiragana di buat oleh para wanita Jepang untuk mempermudah tulisan kanji. Terdapat 46 huruf dasar.
Contoh hurufnya:
- なないろ (Nanairo): 7 macam / 7 warna

❇ Katakana
↪Huruf ini ialah huruf untuk serap asing. Secara sejarah, huruf ini di buat oleh para lelaki Jepang, maka dari itu penulisna nya sedikit lebih tegas. Fungsi huruf ini ialah untuk menulis nama orang, negara, wilayah, wisata alam, rekreasi, produk luar Jepang yang semisalkan "diubah" ke Bahasa Jepang tetapi agak "memaksa". Terdapat 46 huruf dasar.
Contoh hurufnya:
↪コンピューター (Konpyuutaa): Computer

❇ Romaji
↪Roma di ambil dari kata latin, ji adalah huruf. Berarti mengartikan: huruf latin. Biasanya ini berfungsi bagi yang gak bisa baca hiragana, katakana, ataupun kanji
Contoh huruf:
↪Nagai Hikari: cahaya Panjang

❇ Furigana
↪Huruf ini sebenarnya terdiri dari hiragana. Tetapi, maksud dari furigana ini adalah "teks kecil". Di setiap buku Jepang, ada keterangan furigana di atas kanji dengan ukuran keci. Fungsinya agar seseorang yang tak bisa membaca kanji dapat membaca buku atau teks tersebut.
