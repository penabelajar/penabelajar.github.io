15. Temperatur suhu ruangan 27 di bawah nol .Tentukan temperatur suhu ruangan tersebut!​


Jawab:

-27°

Penjelasan dengan langkah-langkah:

Karena dibawah nol, maka nilai tersebut merupakan bilangan negatif. maka :

Temperatur suhu ruangan 27 dibawah nol = -27°

⏩ Jawaban:

-27°

⏩ Penyelesaian soal:

= 0 - 27

= -27

Catatan:

Bilangan positif ( + ) terletak di sebelah kanan angka 0

Bilangan negatif ( - ) Terletak di sebelah kiri angka 0
