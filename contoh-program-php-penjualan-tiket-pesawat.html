misal: jumlah tiket anak" =A   dan    jumlah tiket dewasa= D
A + D = 180        dan       20 A + 30 D = 4200
A= 180-D ⇒ 20 (180-D) + 30D=4200
                   3600-20D+30D   =4200
                                   10D   =600 ⇒ D=60
D=60 ⇒ A=180-D=180-60=120
jadi, tiket untuk anak" terjual sebanyak 120 tiket, dan tiket untuk dewasa sebanyak 60 tiket

Pesawat penumpang mempunyai tempat duduk sebanyak 48 kursi. setiap penumpang kelas utama boleh membawa bagasi sebanyak 60 kg, sedangkan kelas ekonomi 20 kg. pesawat hanya dapat membawa bagasi sebanyak 1.440 kg. Harga tiket kelas utama adalah Rp 1.500.000 dan kelas ekonomi Rp 1.000.000dari permasalahan di atas pernyataan yang tepat adalah ... .Pendapatan Penjualan tiket minimum jika menjual tiket kelas ekonomi sebanyak 48Pendapatan Penjualan tiket maksimum sebesar Rp 36.000.000Pendapatan Penjualan tiket maksimum jika menjual tiket kelas utama sebanyak 24Pendapatan Penjualan tiket maksimum jika menjual tiket kelas ekonomi sebanyak 36Pendapatan Penjualan tiket minimum sebesar Rp 54.000.000​


Penjelasan dengan langkah-langkah:

• Pesawat memiliki 48 kursi

• 1 Penumpang kelas utama : bagasi 60 kg

• 1 Penumpang kelas ekonomi : bagasi 20 kg

• Pesawat dapat membawa : bagasi 1440 kg

• Harga tiket kelas utama : 150.000

• Harga tiket kelas ekonomi : 100.000

Kita buat persamaan Matematika,

misalkan,

- Kursi pesawat = Kp

- Penumpang kelas utama = Pku

- Penumpang kelas ekonomi = Pke

- Bagasi kelas utama = Bku

- Bagasi kelas ekononmi = Bku

- Jumlah bagasi Maksimum pesawat = Bpm

- Harga tiket kelas utama = Hku

- Harga tiket kelas ekonomi = Hke

• Kp = 48

• Pku + Pke = Kp

• Pku + Pke = 48 ... pers 1

• Bpm = 1440

• 60Bku + 20Bke = Bpm

• 60Bku + 20Bke = 1440

• 1 Pku = 60Bku

• 1 Pke = 20Bke

• 60Pku + 20Pke = 1440 ... pers 2

60Pku + 20Pke = 1440 ... pers 2• nPku (1.500.000) + nPke (1.000.000) = .... pers 3 (Hasil nilai yang optimum)

Untuk menetukan nilai Optimum, maka kita dapat menentukan nilai setiap nPku dan nPke

• Pku max, syarat Pke = 0

60Pku + 20Pke = 1440

60Pku = 1440

Pku = 24

HP = {nPku, nPke} = {24, 0} ... Hasil 1

• Pke max, syarat Pku = 0

60Pku + 20Pke = 1440

20Pke = 1440

Pku = 72

HP = {nPku, nPke} = {0, 72} ... Hasil 2

• Pke = 48,

Pku + Pke = 48

Pku + 48 = 48

Pku = 0

HP = {nPku, nPke} = {0, 48} ... Hasil 3

• eliminasi Pke dengan pers 1 dikali 20 dan pers 1 dikali 1

20Pku + 20Pke = 960

60Pku + 20Pke = 1440

__________________-

-40Pku = -480

Pku = 12

• Substitusi nilai Pku ke pers 1

Pku + Pke = 48

12 + Pke = 48

Pke = 36

HP = { nPku, nPke } = {12, 36} ... Hasil 4

Kita cari hasil optimum dengan memasukkan nilai nPku dan nPke pada pers 3

• Hasil 1 {nPku, nPke} = {24, 0}

nPku (1.500.000) + nPke (1.000.000)

= (24)(1.500.000) + (0)(1.000.000)

= 36.000.000

• Hasil 2 {nPku, nPke} = {0, 72}

nPku (1.500.000) + nPke (1.000.000)

= (0)(1.500.000) + (72)(1.000.000)

= 72.000.000

• Hasil 3 {nPku, nPke} = {0, 48}

nPku (1.500.000) + nPke (1.000.000)

= (0)(1.500.000) + (48)(1.000.000)

= 48.000.000

• Hasil 4 {nPku, nPke} = {12, 36}

nPku (1.500.000) + nPke (1.000.000)

= (12)(1.500.000) + (36)(1.000.000)

= 18.000.000 + 36.000.000

= 54.000.000

• Penjualan tiket kelas ekonomi minimum sebanyak 48

= Rp. 48.000.000

• Pendapatan Maksimum Rp. 36.000.000 dengan menjual 24 tiket kelas utama dan 0 tiket kelas ekonomi

• Penjualan tiket kelas utama maksimum sebanyak 24 = Rp. 36.000.000

• Penjualan tiket kelas ekonomi maksimum sebanyak 36 = Rp. 54.000.000

• Pendapatan Minimum Rp. 54.000.000 dengan menjual 12 tiket kelas utama dan 36 tiket kelas ekonomi
