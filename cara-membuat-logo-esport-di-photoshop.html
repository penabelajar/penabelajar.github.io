15. bagaimana cara membuat photoshop di komputer


download aplikasinya, cari langkah" mendownload aplikasi di youtubeCara menginstal program Adobe Photoshop:

1. Download software Adobe Photoshop dari situs ini : http://www.adobe.com/products/photoshop/free-trial-download.html

2.  Buka folder Setup dan jalankan program Setup.exe dari hasil download.

3. Buka Adobe Photoshop CS3 yang anda mau lakukan instal. Buka folder yang anda diinginkan dan klik Next.

4. Lalu Klik Finish.

5. Tunggu loading pada beberapa menit.

6. Terima Perjanjian Lisensi Pengguna Akhir (EULA), mengambil dicatat bahwa Anda melanggar setiap kata itu. Klik Agree.

7. Pilih lokasi penginstalan yang anda inginkan, lalu pilih Next

8. Klik Install, akan memakan waktu cukup lama.

9. Tunggu loading pada beberapa menit.

10. Install selesai kemudian klik finish.

11. Cari serial number di internet dan buka program Adobe Photoshop.

12. Ketik serial number ke menu help --> register product (internet matikan)

13. Selesai
