8. apa keunggulan dan kelemahan dari format EPS​


Jawaban:

KEKURANGAN FILE EPS

Sebagai salah satu format berkas yang banyak digunakan oleh desainer grafis untuk keperluan percetakan, EPS mempunyai beberapa kekurangan yang cukup menjadi perhatian. Format file ini diketahui tidak mampu menyimpan informasi alpha channel. Oleh karena itu, banyak pengguna yang menggunakan format EPS apabila sudah berada pada tahap final pra cetak.

KELEBIHAN FORMAT FILE EPS

Terlepas dari kekurangan yang dimiliki, format EPS mempunyai kelebihan yang membuatnya banyak dipilih dalam dunia desain grafis. Pertama, format file EPS menggunakan PostScript yang dikenali oleh sebagian besar program pra cetak. Tidak heran jika format ini sering digunakan pada tahap akhir pembuatan desain.

Berkas dengan format EPS biasanya mempunyai isi dengan format teks polos. Hal ini menjadi keunggulan tersendiri karena membuat pengguna merasa nyaman ketika mengakses file tersebut. Selain itu, EPS juga mempunyai gambar pratayang yang memudahkan program aplikasi untuk melihat isi dalam file tanpa perlu melakukan proses render.

Di samping itu, berkas EPS juga sudah mampu menyimpan beragam mode warna yang meliputi CMYK, Lab, RGB, grayscale, dan bitmap. Kamu juga bisa menyimpan clipping path menggunakan format EPS ini. Untuk membuka, mengedit, mengkonversi, dan menyimpan berkas EPS, kamu bisa menggunakan beberapa program seperti Adobe Photoshop, CorelDraw, Adobe Illustrator, dan lain sebagainya.

Itu tadi pembahasan tentang Pengertian EPS Beserta Kelebihan dan Kekurangan, semoga bermanfaat dan dapat menambah wawasan kita.

Umur Naruto : 40 Tahun

Penjelasan dengan langkah-langkah:

Perbandingan Sazuke : Naruto = 9 : 8

Selisih perbandingan = 9 - 8 = 1

Selisih umur = 5 tahun

Umur Sazuke

= 9/1 × 5 tahun

= 45 tahun

Umur Naruto

= 8/1 × 5 tahun

= 40 tahun
