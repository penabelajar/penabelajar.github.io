Jawaban:

// Example program

#include <iostream>

#include <string>

using namespace std;

int main()

{

   int gajiperjam = 500,

       gajiharian = 0,

       jamkerja = 0,

       jambonus = 7;

       

   cout << "Gaji Perjam : " << gajiperjam << "\n";

   cout << "Masukan Jumlah/Lama Jam Kerja : "; cin >> jamkerja;

   

   gajiharian = jamkerja * gajiperjam;

   cout << "- Gaji Pokok : " << gajiharian << "\n";

   

   switch(jamkerja > jambonus) {

       case true : {

           int jamsisa = jamkerja - jambonus;

           int gajilembur = (int)(jamsisa * 1.5 * gajiperjam);

           gajiharian += gajilembur;

           cout << "- Gaji Lembur : " << gajilembur << "\n";

       }

   }

   

   cout << "- Total Gaji Harian Pegawai : " << gajiharian;

}

Perusahaan ingin membuat program untuk menghitung gaji karyawannya. Maka rancanglah algoritma nyaJawabanPendahuluan

Algoritma adalah susunan logis dan sistematis dari langkah-langkah untuk memecahkan atau menyelesaikan suatu permasalahan. Misalnya adalah langkah – langkah untuk memasak mie instan, langkah – langkah yang dapat kita temukan di balik bungkus mie instan tersebut juga dapat kita sebut sebagai sebuah algoritma karena merupakan langkah-langkah untuk menyelesaikan permasalahan yaitu memasak mi instan menjadi matang.

Dalam dunia pemrograman, algoritma digunakan untuk membuat software dengan berbagai kegunaan yang digunakan pada perangkat komputer. Algoritma dalam pembuatan software diimplementasikan dengan beberapa bahasa pemrograman seperti C/C++, C#, Visual Basic, dan bahasa pemrograman lainnya.Tanpa algoritma yang tepat, baris demi baris kode tidak akan menghasilkan output yang diinginkan. Oleh karena itu algortima sangat penting dalam membuat program berfungsi sebagaimana mestinya.

Pembahasan

Semakin kompleks suatu algoritma, semakin banyak pula statement di dalamnya dan tidak selamanya berurutan. Statement-statement ini perlu diseleksi dan disesuaikan dengan permasalahan dan prosedur menyelesaikan suatu masalah. Proses penyeleksian statement inilah yang disebut dengan percabangan.

Sebuah perusahaan memiliki karyawan 3 golongan yaitu a b dan c . Perjam nya golongan a digaji 19.500 ,b 21.000, c 25.000. lembur dibayar 30.000 . 1 hari memiliki jam kerja 8 jam dan seminggu terdapat 6 hari jam kerja.

Pada informasi diatas, kita mengetahui ada 3 jenis karyawan yang berbeda dan memiliki gaji masing-masing. jam kerja juga diinput.

Itu berarti yang menjadi variabel disini adalah jenis karyawan. Gaji untuk masing-masing karyawan sudah diketahui sebelumnya. Inti dari algoritma ini adalah menentukan input jenis karyawan kemudian menentukan berapa gaji berdasarkan jenis karyawan tersebut. Disini kita akan gunakan percabangan yang merupakan perintah decision atau pengambil kepututusan terhadap suatu kondisi untuk memilih proses yang sesuai.

Kesimpulan

Maka kita bisa buat algoritma yang akan dituliskan dalam pseudocode berikut ini.

ALGORITMA menentukan gaji karyawan

Deklarasi:

tipe karyawan, gajiA, gajiB, gajiC, gajiLembur, gajiTotal, jam kerja, lembur: integer

gajiA = 19500

gajiB = 21000

gajiC = 25000

gajiLembur = 30000

Deskripsi:

Read(tipe karyawan)

Read(jam kerja)

if jam kerja > 48 then

        lembur = jam kerja - 48

else lembur = 0

if tipe karyawan = A then gajiTotal = 48 * gajiA

else if tipe karyawan = B then gajiTotal = 48 * gajiB

else if tipe karyawan = C then gajiTotal = 48 * gajiC

gajiTotal = gajiTotal + lebuh * 30000

Write(gajiTotal)

Flowchart untuk soal diatas ada di lampiran.

Untuk lebih lanjut, mari kita baca pengertian dan jenis-jenis flowchart.

Flowchart adalah adalah suatu gambaran alur sebuah proses yang berurutan yang digambarkan dengan simbol-simbol tertentu.

Ada lima jenis Flowchart berfasarkan fungsinya:

1. Flowchart Sistem (System Flowchart)

Flowchar yang menggambarkan urutan kerja di dalam keseluruhan sistem dan menjelaskan urutan dari prosedur-prosedur di dalam sistem tersebut. Flowchart sistem merupakan dekripsi grafis dari urutan prosedur-prosedur yang disusun sehingga membentuk sebuah sistem yang lengkap.

2. Flowchart Dokumen (Document Flowchart)

Document flowchart atau sering juga disebut bagan alir formulir (form flowchart) dan paperwork flowchart merupakan flowchar yang menunjukkan urutan dari laporan dan formulir beserta tembusan-tembusannya. Flowchart ini menggunakan gambar dan simbol seperti yang digunakan di dalam Flowchart sistem.

3. Flowchart Skematik (Schematic Flowchart)

Schematic flowchart sebenarnya mirip dengan flowchart sistem, fungsinya adalah menggambarkan urutan prosedur di dalam suatu sistem. yang menjadi pembedanya adalah, flowchart skematik juga menggunakan gambar-gambar alat secara fisik yang bertujuan untuk memudahkan pembaca yang tidak terlalu paham dengan simbol-simbol flowchart.

4. Flowchart Program (Program Flowchart)

Program flowchart merupakan flowchart yang menjelaskan secara rinci algoritma atau urutan-urutan dari jalanya program.

5. Flowchart Proses (Process Flowchart)

Flowchart Proses merupakan cara penggambaran proses dalam ranah industrial yang berisi langkah-langkah secara berurutan dalam suatu prosedur atau sistem. Flowchart ini memiliki simbol-simbol khusus
