9. tolong bantu jawab soal limit kelas XII​


Jawab:

limit tak hingga

Penjelasan dengan langkah-langkah:

lim(x-> ∞)  [√(x + 2)  - √(x + 4) ]

= lim (x-> ∞) [√(x/x +2/x) - √(x/x + 4/x) ]

= lim (x-> ∞) [√(1 +0) - √(1 + 0) ]

= 1 - 1 = 0

..

lim(x-> ∞) [ √(5 - 4x + 3x²) - √(4 - 3x + 3x²)  ] /(2x)

= lim(x-> ∞) √(5/x² - 4x/x²  + 3x²/x² ) +√(4/x²  - 3x/x²  + 3x²/x² )  ] /(2x)/(x)

= lim(x-> ∞) √(0 -0  + 3) + √(0-0+ 3)  ] /(2)

= (√3 + √3)/(2)

= 2√3 / 2

= √3
