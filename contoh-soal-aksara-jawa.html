Jawaban dan Pembahasan

4. Aksara Latin : TERANG TRAWACA.

Tegese : Terang benderang.

Ukara : Lampu bangjo ing alun - alun kae cahyane terang trawaca.

5. Aksara Latin : GAGAH PRAKOSA.

Tegese : Awake kuwat lan gagah.

Ukara : Mas Doni awake gagah prakosa amarga kerep olahraga sing abot-abot.

6. Aksara Latin : SANAK KADANG.

Tegese : Saudara.

Ukara : Aku nduwe sanak kadang adoh sing manggon ing Jakarta.

7. Aksara Latin : SEKTI MANDRAGUNA.

Tegese : Sekti banget.

Ukara : Ing crita pewayangan, Raden Gatotkaca iku nduweni kadigdayan kang sekti mandraguna.

8. Aksara Latin : GULUNG KOMING.

Tegese : Jungkir walik.

Ukara : Nalika pelajaran Penjaskes, aku praktek kaya wong gulung koming.

9. Aksara Latin : BABAK BUNDHAS.

Tegese : Lecet kabeh awake.

Ukara : Paijo sikile babak bundhas amarga ora ngati-ati, banjur tiba saka sepedha.

10. Aksara Latin : MURUB MUBYAR.

Tegese : Sorote werna-werni, padhang banget.

Ukara : Nalika Agustusan, kampung ku padha masang lampu kang murub mubyar.

Penjelasan

Tembung entar yaiku tembung loro utawa luwih sing digabung dadi siji lan tegese dadi beda saka asal-usule.

Tembung entar tegese ora kaya teges salugune (kata kiasan) utawa tembung kang ora kena ditegesi sawantahe bae. Ing basa Indonesia diarani kata kiasan.

Tuladhane:

adol ayu = ngendelake ayune

adol bagus = ngendelake baguse

adol gawe = ngatonake pegaweyane

adol kringet = nyambut gawe

kandel kupinge = ora nggugu pitutur

kaku atine = tansah ora sarujuk/sulaya

kasar tembunge = tembunge saru
