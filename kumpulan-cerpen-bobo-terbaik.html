8. Bantu helpppppppp mau bobo:( 3√24+2√3(√32-2√18) Makacih kakak" Yang baik


Jawab :

3√24 + 2√3[√32 - 2√18]

= 3√(4 x 6) + 2√3[√(16 x 2) - 2√(9 x 2)]

= 3 x 2√6 + 2√3[4√2 - 2 x 3√2]

= 6√6 + 2√3 x [4√2 - 6√2]

= 6√6 + 2√3 x [-2√2]

= 6√6 - 4√6

= 2√6
