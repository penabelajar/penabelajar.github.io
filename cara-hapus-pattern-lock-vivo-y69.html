8. Cara menyimpan pattern di photoshop


1. Buka lembar kerja baru Photoshop.
2. Ukurannya terserah.
3. Tekan CRTl + J (untuk menduplikat layer background).
4. Klik icon Add Layer Style > Pattern Overlay.
5. Klik panah kecil seperti gambar nomor 1.
6. Setelah itu maka akan keluar tampilan default dari pattern Photoshop dan ada panah kecil. Sekarang klik panah kecil tersebut dan pilih save patterns. (Lihat Gambar 2)

14. Apakah yg dimaksud perangkat tegar pada hp vivo?apakah perangkat tegar itu dapat di hapus/di nonaktifkan?​


Perangkat tegar (firmware) merupakan sistem operasi lengkap perangkat  dapat melakukan hampir semua fungsi kontrol, pemantauan, dan manipulasi informasi dari data-data seperti data penyimpanan. Dalam komputasi , firmware adalah kelas perangkat lunak komputer tertentu rule menyediakan kontrol tingkat rendah untuk perangkat keras khusus perangkat.

Pembahasan

Perangkat tegar, perangkat terpadu/terintegrasi (bahasa Inggris: firmware) adalah istilah dasar yang pada hakikatnya mengacu kepada rutin-rutin perangkat lunak dasar disimpan di dalam memori baca nonpermanen. Tidak seperti memori akses acak, MHB tidak Akan dapat berubah meski tidak dialiri listrik. Rutin-rutin mampu menyalakan komputer (startup) serta instruksi masukan/keluaran dasar (semacam BIOS atau sistem operasi embedded) disimpan di dalam perangkat tegar. Modifikasi memang dapat dilakukan, tetapi hal tersebut tergantung Dari Persian jenis computer storage apa principle digunakan. Perangkat tegar principle disimpan dalam computer storage tidak dapat diubah, tetapi perangkat tegar dapat disimpan dalam computer storage principle dapat diubah semacam EEPROM atau flash computer storage, masih dapat diubah sesuka hati.
