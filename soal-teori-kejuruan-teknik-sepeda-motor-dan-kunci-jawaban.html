Susunan dilakukan secara melingkar, sehingga dihitung dengan permutas siklis:

P siklis = (n-1)!

= (5-1)!

= 4!

= 4 x 3 x 2 x 1

= 24

Jadi, ada 24 cara.

7. kak tolong bantu menjawabnyaa, soal: pada hari minggu ditempat lahan parkir seorang juru parkir menghitung jumlah kendaraan mobil ditambah sepeda motor adalah 75 buah kendaraan,sedangkan jumlah semua roda-rodanya adalah 200 buah roda. jika uang sewa parkir untuk mobil Rp3.000,00 sedangkan untuk sepeda motor Rp2.000,00. Tentukan uang sewa parkir yg diperoleh oleh juru parkir pada hari minggu! ​


sewa parkir mobil = Rp 3.000,00

sewa parkir sepeda motor = Rp 2.000,00

misalkan :

mobil = a

sepeda motor = b

maka :

a + b = 75 buah kendaraan

4a + 2b = 200 buah roda

a + b = 75

a = 75 - b

4a + 2b = 200

4(75 - b) + 2b = 200

300 - 4b + 2b = 200

300 - 2b = 200

2b = 300 - 200

2b = 100

b = 100/2

b = 50

a = 75 - b

a = 75 - 50

a = 25

uang sewa parkir yang diperoleh juru parkir pada hari minggu

= 3.000a + 2.000b

= 3.000(25) + 2.000(50)

= 75.000 + 100.000

= Rp 175.000,00
