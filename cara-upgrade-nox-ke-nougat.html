2. Cara Mengupradge Android Jelly Bean Ke Nougat?


mungkin mengupgrade dari Android Jelly Bean ke Nougat tak semudah itu karena perlu waktu yang lama. jadi untuk mengupgradenya perlu:
1. koneksi internet stabil dan memadai
2. dukungan Android dari pembuat ponsel
3. dukungan upgrade lewat udara
4. WI-FI yang stabil koneksinya
5. baterai minimal 50% keatas
6. data wajib dibackup

jadi disini akan menjelaskan tentang upgrade melalui udara:
1. ke setelan => tekan pembaruan nirkabel => tekan check for updates
2. jika ada update terbaru, file akan terunduh. dan jika file sudah terunduh, kita akan diminta untuk menghidupkan ulang hp
3. ketika berada didalam update, tunggu hingga selesai dan jangan matikan ponsel. jika terjadi, ponsel akan mengalami kerusakan software (bootloop/brick)

kalau tidak bisa, dapat menggunakan file yang diunduh dari internet
1. jika updatenya pakai hp (langsung), pergi ke pilihan yang sama
2. kelocal updates lalu pilih file
3. jika ada peringatan seperti digambar kedua, pastikan bahwa file sudah benar dan lakukan update sekarang
4. tunggu hingga update selesai

4. cara meng upgrade ios


Ada 2 cara yg bisa dipakai untuk uprade ios:
1. Langsung dari perangkat ios
Caranya:
- Pastikan perangkat sudah terhubung ke internet (saran, pakai wi-fi karena akan memakan banyak data) dan perangkat mempunyai space memori minimal 1,6 GB atau tergantung besar size os ios yg ingin diupgrade
- Kemudian, masuk ke pengaturan (setting), lalu ke umum (general), kemudian ke pembaharuan perangkat lunak (software update)
- Kemudian, jika ada pembaharuan/update, pilih Unduh dan Instal (download and install) dan tunggu hingga selesai (kalau diminta password hp, masukkan saja passwordnya)
2. Lewat Itunes
Caranya:
- Hubungkan perangkat ke pc yang sudah terinstal itunes
- Kemudian, tekan ikon Iphone atau Ipad di Itunes
- Kemudian, klik Check for Update lalu pilih Download and Update
Semoga membantu...

5. bagaimana cara mengupgrade windows 10


Cara ini hanya berlaku untuk Windows 7 dan Windows 8/8.1. Untuk pengguna Windows Vista dan Windows XP, harus tetap mendownload file ISO Windows 10 dari situs resmi Microsoft.
1. Download sebuah program kecil dari Microsoft
2. Jalankan program tesebut sebagai Admin lalu tunggu hingga proses selesai dan jika sudah, restart komputer kalian.
3. Nah jika sudah sekarang kalian nyalakan koneksi Internet kalian dan pergi ke Windows Update. Tunggu proses untuk beberapa menit ( kurang lebih 5 sampai 10 menit tergantung koneksi Internet kalian ), nanti kalian akan diberi informasi tentang Windows 10 yang akan diinstall di PC kalian.
4. Contohnya adalah gambar dibawah ini saat saya memutuskan untuk mencoba upgrade Windows 8.1 milik saya menjadi Windows 10. Untuk memulai proses upgrade, tentu saja tinggal klik Install dan tunggu hingga proses upgrade selesai.
