PROSES MEMBUAT LAYOUT

1. KONSEP DESAIN

Bagaimana mendesain layout yang baik? Pertanyaan berikut adalah langkah awal…

Apa tujuan desain tersebut?
–Siapa target audiens nya?
–Apa pesan yang ingin disampaikan?
–Bagaimana cara penyampaiannya?
Dimana, di media apa dan kapan desain itu akan dilihat oleh target audiens?
2. MEDIA DAN SPESIFIKASINYA

Media apa yang paling cocok: Flier, brosur 3 lipatan, spanduk, plasma screen, balon udara, dll
Bahan: Kertas fancy, kertas daur ulang, kain, dll
Ukuran :A4, A3, 160x60cm untuk x-banner, dll
Posisi: A4 tegak (portrait) atau mendatar (landscape)
Kapan, Berapa lama, dan dimana saja karya desain tersebut akan didistribusikan ke target audiens.
3. THUMBNAILS DAN DUMMY

Thumbnails = sketsa layout dalam bentuk mini, tidak dengan menggunakan komputer (sebaiknya).
–Fungsi: memperkirakan letak elemen-elemen layout pada suatu halaman tunggal, juga urutan-urutan pengaturan halaman desain publikasi yang kompleks.

Dummy = contoh jadi suatu desain nantinya (mock-up).
4. PERCETAKAN

Offset: teknik yang paling populer untuk mencetak brosur, buku majalah, tabloid, koran, kalender, dll
Flexografi/ cetak tinggi: mencetak di atas karton gelombang atau label kemasan produk.
Rotogravure: mencetak label berbahan plastik.
Sablon/cetak saring/screen printing: mencetak kaos, mug, kartu nama.
Digital: mencetak dalam waktu singkat dengan kuantitas tidak terlalu besar, seperti banner, poster, dll.

Di dalam menu layout terdapat beberapa menu pilihan diantaraya sebagai berikut dan fungsinya dari masing - masing menu tersebut yaitu

Menu Layout

1. Insert Page yaitu digunakan untuk menambahkan page/halaman pada lembar kerja yang sama pada CorelDraw

2. Duplicate Page digunakan untuk menyalin halaman sebelumnya ke halaman berikutnya tanpa menghilangkan halaman yang disalin

3. Rename Page berfungsi untuk mengganti atau merubah naman halaman yang ada pada lembar kerja CorelDraw.

4. Delete Page berfungsi untuk menghilangkan halaman atau menghapus halaman.

5. Go to Page digunakan untuk cara pintas yaitu untuk menuju halaman yang dituju misalanya dari halaman 1 kehalaman 2 atau langsung ke halaman 3,4, 5 dan seterusnya, sesui dengan kubutahan anda.

6. Insert Page Number berfungsi untuk menambahakan halaman baru di lembar kerja CorelDraw.

7. Page Number Setting digunakan sebagai pengaturan pada halaman yang ada pada CorelDraw.

8. Swicth Page Orientation berfungsi untuk merubah Page Border dari Portrait ke Landscape atau sebaliknya.Page Setup

9. Page Background berfungsi untuk memberi warna pada Page Border atau menghilangkan warna pada Page Background.

10. Page Layout berfungsi sebagai pengaturan pada konsep letak pada lembar kerja.

Tool Box

Tool Box adalah suatu kotak atau alat yang memiliki fungsi untuk menggambar, modifikasi objek, memberi warna, menulis teks. Tool Box memiliki beberapa fungsi di satu kotak atau tempat yang sama, jika ingin membuka atau mempergunakan fungi lain maka silahkan klik tanda segitiga kecil di pojok kanan bawah salah satu tool box.

Tool Bar

Tool Bar merupakan sekumpulan tombol shortcut yang memiliki fungsi untuk menjalankan perintah dengan cepat. ShortCut Ini adalah bagian yang paling sering digunakan oleh desainer grafis dalam melakukan sesuatu. CorelDraw memberikan fitur dalam toolbar yang powerfull namun mudah sekali digunakan.

Menu Bar

Menu Bar berisi tombol-tombol text yang memiliki banyak sekali fungsi yang berbeda sesuai dengan namanya masing-masing. Desainer pun sering menggunakan fasilitas ini biasanya dalam memberikan suatu effect berhubungan dengan vector dan bitmap.

Property Bar

Property Bar berisi sekumpulan tombol yang isinya itu menyesuaikan dengan tools yang sedang kita pilih di dalam tool box.

Docker

Docker merupakan kotak dialog yang di dalamnya berisi berbagai tombol perintah untuk mengatur, memberi effect kepada sebuah objek maupun teks.

Rulers

Rulers merupakan alat ukur berupa penggaris dibagian atas dari drawing page, dan di bagian kanan drawing page, rullers digunakan sebagai acuan yang mampu menunjukkan ukuran suatu objek atau dokumen, serta tata letak suatu desain.

Status Bar

Status Bar terletak dibagian bagian bawah area program sebelah kanan. Status Bar berisi informasi mengenai objek (teks) seperti ukuran, bentuk, warna, outline, format gambar, jenis huruf beserta  ukurannya.

Drawing Page

Drawing Page adalah adalah sebuah area kerja utama di dalam CorelDraw di mana Anda dapat mendesain suatu objek di dalamnya dan merupakan area cetak suatu desain.

Color Palette

Color Palette bagian dari CorelDraw yang berisi koleksi warna di dalam sebuah bar dan secara default/bawaan terletak di sebelah kanan dokumen / program.

14. Langkah langkah cara membuat layout text seprerti karan atau majalah


1. Buka fie dokumen Microsoft Word anda yang sudah ada artikelnya.

2. Selanjutnya yaitu membuat tulisan pada artikel menjadi dua kolom caranya yaitu Pilih tulisan anda dari kata pertama sampai kata terakhir, lalu pilih Tab Menu Page Layout -> Columns -> Two.

3. Maka tulisan artikel anda akan menjadi dua kolom.  Selanjutnya yaitu membuat kata awal hurufnya menjadi besar.  Caranya yaitu pilih Tab Menu Insert -> Drop Cap.

4. Selanjutnya silahkan anda masukkan sebuah gambar dengan cara pilih menu INSERT -> PICTURE.  Lalu setelah gambar sudah berada di artikel anda silahkan pilih Tab Manu Format ->Position -> With Text Wrapping.
