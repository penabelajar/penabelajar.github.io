Jawaban

Perbedaan dari Onii-chan dan Onee-chan adalah :

Onii-chan = Panggilan akrab untuk kakak laki-lakiOnee-chan = Panggilan akrab untuk kakak perempuan

Sedangkan arti dari Onii-chan daisuki adalah "Aku mencintai kakak (laki-laki)"

Penjelasan

Apa itu chan?

Akhiran -chan merupakan akhiran panggilan yang digunakan dengan tujuan untuk memberi kesan imut. Biasa digunakan ke anak kecil atau bisa juga ke pacar perempuan. Kadang juga digunakan ke saudara kandung untuk panggilan yang akrab.

Apa itu daisuki?

Daisuki dalam kanji ditulis 大好き (Daisuki), yang dimana 大 (Dai) berarti "besar", dan 好き (Suki) berarti "suka". Kalau diartikan jadi seperti rasa suka yang besar atau sangat suka. Kata ini bisa digunakan ke orang, benda, kegiatan, dll. Kalau digunakan ke orang, bisa berarti "cinta".

3. Arti onii san dan onii chan


Arti dari お兄さん (oniisan) dan お兄ちゃん (oniichan), saling memiliki arti yang sama yaitu kakak lelaki atau artinya bisa berarti abang.

Pembahasan

Walau pada dasarnya arti antara お兄さん (oniisan) dan お兄ちゃん (oniichan) memang memiliki arti yang sama, namun memiliki letak perbedaannya. Untuk mengetahui letak perbedaannya bisa dilihat sebagai berikut :

お兄さん (oniisan), umumnya bisa digunakan untuk memanggil/panggilan bagi kakak lelaki sendiri. Namun ketika kita ingin menyebutkan kakak lelaki orang lain yang kurang akrab/dekat, atau sedang membicarakan kakak orang lain umumnya digunakan kata oniisan agar sopan.

お兄ちゃん (oniichan) bisa digunakan terhadap kakak lelaki kita sendiri maupun juga kakak lelaki yang lain, dengan syarat jika kakak lelaki orang lain termasuk orang yang dekat/akrab dengan kita.

Note :

Kata お兄さん (oniisan) atau お兄ちゃん (oniichan) tidak hanya bisa dipanggil untuk kakak lelaki sendiri, namun juga bisa kita pakai panggilan ini untuk kakak kelas kita, terhadap kakak sepupu, bahkan bisa panggilan terhadap kakak lelaki dari sahabat atau kerabat kita.

9. Apa artinya onii chan janaiyo


Jawaban :

Arti dari 'Onii-chan janai yo' adalah "(Aku) bukan kakak (laki-laki)(mu)".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Kalimat ini akan saya bagi menjadi 3 pembahasannya, yakni 'Onii-chan', 'Janai', dan 'Yo'. Simak pembahasannya sebagai berikut.

1. お兄ちゃん / おにいちゃん / Onii-chan

Berarti "Kakak laki-laki". Panggilan kakak laki-laki yang satu ini menggunakan akhiran 'Chan'. Gunanya adalah sebagai panggilan yang akrab, juga terkesan manja dan imut. Biasa digunakan oleh anak-anak kecil untuk memanggil kakak nya. Selain menggunakan 'Chan', kalian bisa menggunakan 'San' untuk yang sopannya, dan 'Sama' untuk yang sangat sangat sopannya.

2. じゃない / Janai

Berarti "Bukan" atau "Nggak" (tergantung konteks). Kata ini biasa digunakan untuk mengubah suatu kata benda dan kata sifat-na menjadi bentuk negatif. Kata ini termasuk ke ungkapan kasual, tapi bisa diubah menjadi sopan dengan menambahkan 'Desu' setelahnya. Selain itu juga bisa dengan mengganti 'Ja' dengan 'Dewa', menjadi 'Dewanai'. Atau bisa juga dengan mengganti 'Nai' dengan 'Arimasen'.

3. よ / Yo

Tidak ada arti khusus, kadang bisa diartikan sebagai "Loh", "Woi", "Dong", "Kok", dll. Kata ini merupakan partikel atau kata bantu kalimat, yang salah satu fungsinya untuk menegaskan suatu kalimat tersebut. Partikel ini diletakkan di akhir kalimat, maka dari itu tergolong ke dalam 終助詞 (Shuujoshi).

Jadi kalimat 'Onii-chan janai yo' ini biasa diucapkan saat ada seseorang yang bukan merupakan adikmu, tapi memanggilmu dengan Onii-chan (kakak laki-laki). Kalau diartikan seperti "Gw bukan kakak lu woi", atau lebih sopannya "Aku bukan kakakmu".

Sedikit tambahan, sebenarnya dalam kalimatnya tidak diberitahu subjeknya siapa. Hal ini dikarenakan memang orang Jepang tidak sering mengucapkan subjek kalimatnya, jika subjeknya sudah diketahui lawan bicara. Tapi dari melihat kalimatnya, saya simpulkan bahwa subjeknya adalah si pembicara sendiri (Saya).

10. Onii-Chan Daisuki artinya?


Jawaban:

aku menyayangimu kakak (laki laki)

Penjelasan:

onii-chan : kakak (laki laki)

daisuki : daisuki (大好き/だいすき) dan aishiteru (愛してる/あいしてる), memiliki fungsi yang sama, yaitu untuk menyatakan dan mengekspresikan rasa suka, dan cinta terhadap pasangan.

Namun, biasanya orang Jepang pada umumnya tidak mengekspresikan rasa kasih sayang mereka secara terbuka dengan kata-kata. Bukan karena mereka tidak menghargai ungkapan cinta, justru karena mereka menganggapnya sebagai hal yang begitu besar dan penting. Sehingga, mereka lebih suka menunjukkan dengan perbuatan, perlakuan, dan perhatian daripada perkataan.

Kata daisuki secara harfiah artinya adalah "Aku sangat menyukaimu", sedangkan aishiteru artinya adalah "Aku cinta padamu". Kata aishiteru dibandingkan dengan daisuki memiliki makna yang sangat dalam. Karenanya, sesuai dengan kultur Jepang, kalimat tersebut hanya digunakan oleh pasangan yang telah menikah atau yang sudah berkomitmen akan menikah.

Jika mengatakan aishiteru kepada pasangan yang baru jadi pacar, sudah pasti mereka akan merasa risih, sehingga jarang sekali ditemukan kata aishiteru di kalangan remaja. Di kalangan remaja, jika menyukai seseorang pada umumnya memakai kata daisuki.

Misalnya: 君のことが大好きです。 "Kimi no koto ga daisuki desu." Artinya, "Aku sangat menyukai semua tentang kamu."

おにちゃん

Dik Goblin

Penjelasan :

Oni dalam bahasa Jepang berarti Goblin, ogre, atau troll

Sementara honorifik -chan biasanya diberikan untuk anak-anak atau sesuatu yang lucu dan imut

Karena itu, Oni-chan bisa berarti "dik troll, dik ogre, dek goblin dsb"

Kalau maksud kamu yang sering kamu denger di anime, itu adalah "onii-chan" yang berarti "abaang, kakaak, aa～"

"onii-chan" sering digunakan di anime, tetapi di dunia nyata tidak.

Selain onii-chan, untuk memanggil kakak laki-laki yang lebih sering digunakan adalah

Nii-san / Onii-san
