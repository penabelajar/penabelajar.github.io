2. bagaimana cara mengerjakan matematika skala?(kelas 5)


Rumus mencari Skala = Jarak sebenarnya : Jarak pada Peta
Rumus mencari Jarak sebenarnya = Skala x jarak pada peta
Rumus mencari Jarak pada Peta = Jarak sebenarnya : Skala.

Pada umumnya Jarak sebenarnya adalah satuan km.
Pada umumnya Jarak pada peta adalah satuan cm.

12. jarak kota A-B pada peta 5 cm, jika skala 1:3.000.000 berapa jarak sebenarnya?Materi : MatematikaMateri kelas : 5Materi : Skalaplis jawabin ya ​


Jawaban:

DIKETAHUI :

• Jarak pada peta 5 cm

• Skala 1 : 3.000.000

DITANYA ?

Jarak sebenarnya ?

JAWABAN :

Js = Jpp / Skala

Js = 5 / 1 : 3.000.000

Js = 5 × 3.000.000/1

Js = 15.000.000 cm

Js = 150 km

Jarak kota A-B pada peta 5 cm, jika skala 1 : 3.000.000 berapakah jarak sebenarnya?

Jawaban : 150 km

Penyelesaian:

Js = 5 : 1/3.000.000

= 5 x 3.000.000

= 15.000.000 cm

= 150 km

Pembahasan:

Skala menyatakan perbandingan antara jarak dua tempat pada denah atau peta dengan jarak sebenarnya. jika skala pada peta tertulis 1 : 3.000.000, artinya 1 cm jarak pada peta mewakili 3.000.000 cm atau 30 km jarak sebenarnya.

Berikut cara menjadi skala, jarak sebenarnya, dan jarak pada peta.

Skala = Jp/Js

Jp = Js x skala

Js = Jp : skala

keterangan:

Jp = jarak pada peta

Js = jarak sebenarnya
