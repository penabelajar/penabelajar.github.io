8. ubahlah sudut A-50°ke dalam bentuk radiant​


Jawaban

[tex] - 50 \degree = - \frac{5\pi}{18} \\ [/tex]

Pembahasan

Rumus mengubah sudut ke bentuk radian :

[tex]x \degree \to{}x \degree \times \frac{\pi}{180 \degree} \\ [/tex]

.

[tex] = - 50 \degree[/tex]

[tex] = - (50 \degree)[/tex]

[tex] = - (50 \times \frac{\pi}{180} ) \\ [/tex]

[tex] = - ( \frac{5 \cancel0\pi}{18 \cancel0} ) \\ [/tex]

[tex] = - ( \frac{5\pi}{18} ) \\ [/tex]

[tex] = - \frac{5\pi}{18} \\ [/tex]

.

14. contoh batch processing


Contoh dari penggunaan batch processing adalah e-mail dan transaksi batch
processing. Dalam suatu sistem batch processing, transaksi secara individual
dientri melalui peralatan terminal, dilakukan validasi tertentu, dan
ditambahkan ke transaction file yang berisi transaksi lain, dan kemudian
dientri ke dalam sistem secara periodik. Di waktu kemudian, selama siklus
pengolahan berikutnya, transaction file dapat divalidasi lebih lanjut dan
kemudian digunakan untuk meng-up date master file yang berkaitan.Batch processing adalah
susunan dari beberapa file yang di petakan dalam sebuah himpunan pengolahan data, yang akan di eksekusi setelah masing2 komponen saling melengkapi.

contohnya:
email, dan atachment (lampirannya).
file update android dan aplikasinya.
proses loading pada os.
proses loading pada smartphone bbm.
