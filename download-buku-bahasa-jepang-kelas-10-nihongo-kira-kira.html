6. Q. Bahasa Jepang dari “Belajar Bahasa Jepang” ...A. Nihongo O Manabu.B. Nihongo O Narau.C. Nihongo O Benkyou.D. Benkyou O Nihongo.​


Bahasa Jepang dari “Belajar Bahasa Jepang" adalah...

A. Nihongo o manabu.

                     

Penjelasan

Pilihan B saya rasa kurang tepat, karena kata 'narau' digunakan jika mempelajari suatu hal dari seseorang. Hal yang dipelajari biasanya juga berhubungan dengan seni atau keterampilan yang memerlukan teknik dan proses belajar secara berulang-ulang.

Contoh kalimat menggunakan 'narau' ↴

   父から空手を習います。

   = Chichi kara karate o naraimasu.

   = Belajar karate dari ayah.

Pilihan C kurang tepat, karena kata 'benkyou' merupakan kata benda. Kalimat tersebut bisa tepat bila penulisannya menjadi 'Nihongo o benkyou suru'. Kata 'suru' digunakan untuk mengubah kata benda 'benkyou' menjadi kata kerja.

Pilihan D jelas salah, karena susunan kalimatnya yang tidak tepat dan kata 'benkyou' bukanlah kata kerja. Harusnya kata 'benkyou' diganti dengan 'benkyou suru', dan pola kalimat yang tepat harusnya "[Objek] o [Predikat]".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Perbedaan 'Benkyou suru', 'Manabu', dan 'Narau' :

・Benkyou suru = "Belajar" yang lebih fokus ke hal-hal yang berhubungan dengan akademik.

・Manabu = "Belajar" yang cakupannya lebih luas dari 'benkyou suru'.

・Narau = “Belajar” yang digunakan ketika jelas ada seseorang yang mengajarkan kita akan sesuatu, dan biasa berhubungan dengan seni atau keterampilan yang memerlukan teknik dan proses belajar secara berulang-ulang.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Fungsi partikel を (O) :

Menunjukkan objekMenunjukkan tempat yang dilewatiMenunjukkan lokasi yang menjadi titik tolak dari suatu kegiatanMenunjukkan profesi

12. Lengkapi kalimat-kalimat berikut mengunakan partikel yang ada di kolom atas! mohon bantuannya untuk menjawab kakTerimakasih Nihongo Kira-Kirakelas 3/XIIHalaman 82​


Nomor 1

わたしは ジャカルタ (に) すんでいます。ジャカルタ (に) モールや ひろばが あります。よく かぞくと モール (に) いきます。うち (から) モール (に) くるまで 15分ぐらいです。

Saya tinggal (di) Jakarta. (Di) Jakarta ada mall, tanah lapang, dll. Saya sering pergi (ke) mall dengan keluarga. (Dari) rumah sampai tiba (di) mall sekitar 15 menit.

Nomor 2

わたしは よく ともだちと ひろば (で) サッカーを します。ひろばは がっこうの まえ (に) あります。

Saya sering bermain bola dengan teman-teman (di) tanah lapang. Tanah lapang nya ada (di) depan sekolah.

Nomor 3

そぼは いま だいどころ (に) います。だいどころ (で) りょうりを しています。

Nenek sekarang ada (di) dapur. Sedang memasak (di) dapur.

Pembahasan

Dalam kalimat Bahasa Jepang terdapat kata bantu yang memiliki bermacam fungsi, yang disebut dengan partikel atau joshi. Partikel ini jumlahnya ada banyak, dan masing-masing memiliki fungsinya tersendiri.

Partikel dibagi menjadi 4 bagian, yaitu :

Kakujoshi (Partikel penanda kasus)Setsuzokujoshi (Partikel konjungsi)Fukujoshi (Partikel adverbial)Shuujoshi (Partikel akhir kalimat)

Kali ini saya hanya akan membahas tiga partikel yang digunakan sebagai jawaban dari teks di soal. Ketiga partikel tersebut tergolong ke dalam kakujoshi. Penjelasan masing-masing partikel nya adalah sebagai berikut.

1. Partikel に (Ni)

Fungsinya ada banyak. Pada teks-teks di soal, partikel ini digunakan untuk menunjukkan letak dan untuk menunjukkan tempat tujuan.

Contoh pada kalimat berikut.

ジャカルタ に すんでいますJakaruta ni sundeimasu Tinggal di Jakarta

*Partikel Ni di sini berfungsi untuk menunjukkan letak, yaitu Jakarta.

Contoh lainnya sebagai berikut.

モール に いきますMooru ni ikimasu Pergi ke mall

*Partikel Ni di sini berfungsi untuk menunjukkan tempat tujuan, yaitu Mall.

2. Partikel から (Kara)

Salah satu fungsi utama nya yaitu untuk menunjukkan titik awal. Bisa juga diartikan sebagai "dari" dalam Bahasa Indonesia. Biasanya partikel ini sering dipasangkan dengan partikel Made, yang berfungsi untuk menunjukkan titik akhir atau juga dapat diartikan sebagai "sampai".

Contoh pada kalimat berikut.

うち から モール までUchi kara mooru made Dari rumah sampai mall

*Partikel Kara di sini berfungsi untuk menunjukkan titik awal, yaitu Rumah. Sedangkan partikel Made di sini berfungsi untuk menunjukkan titik akhir, yaitu Mall.

3. Partikel で (De)

Fungsinya juga ada banyak. Pada teks-teks di soal, partikel ini fungsinya mirip seperti partikel Ni, yaitu untuk menunjukkan letak. Bedanya, partikel De digunakan untuk menunjukkan letak/tempat melakukan suatu aktivitas.

Contoh pada kalimat berikut.

ひろば で サッカー を しますHiroba de sakkaa o shimasu Bermain bola di tanah lapang

*Partikel De di sini berfungsi untuk menunjukkan tempat melakukan suatu aktivitas (bermain bola), yaitu Tanah lapang.
