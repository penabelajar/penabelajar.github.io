6. Bagaimana cara mengatasi laptop black screen tanpa instal ulang


Masuk ke Safe Mode

Nyalakan laptop;

Sebelum masuk tampilan logo Windows 7, tekan tombol F8, beberapa kali, hingga masuk menu Boot Option Windows 7;

Dari opsi yang ada, pilih Safe Mode;

Setelah tampil safe mode, tunggu sekitar 5-10 menit lalu restart laptop dan nyalakn kembali laptop, langkah ini pernah diuji dan memang jika masalahnya tidak parah bisa menjadikan laptop menyala normal kembali;

11. cara mengatasi laptop yang black screen gimana ya? please bantu ya.


1Masuk ke Safe Mode

Solusi awal yang bisa dilakukan sebagai cara mendapatkan Windows yang baik kembali adalah dengan masuk ke mode aman sistem  atau safe mode windows 7. Langkah untuk masuk ke Safe mode Windows 7 sendiri adalah sbb :

Nyalakan laptop;

Sebelum masuk tampilan logo Windows 7, tekan tombol F8, beberapa kali, hingga masuk menu Boot Option Windows 7;

Dari opsi yang ada, pilih Safe Mode;

Setelah tampil safe mode, tunggu sekitar 5-10 menit lalu restart laptop dan nyalakn kembali laptop, langkah ini pernah diuji dan memang jika masalahnya tidak parah bisa menjadikan laptop menyala normal kembali;

Biasanya masalah seperti ini diakibatkan oleh service Windows yang bermasalah atau timing service yang delay.


2. Instal ulang graphic (display) adapter

Fungsi dari graphic adapter sendiri adalah menampilkan output hasil processing ke layar monitor, dan rendering, meski sekilas tak ada kaitan langsung antara blank screen dengan kerja graphic adapter, tetapi sebenarnya masalah pada driver graphic bisa mengakibatkan sistem menjadi unstable dan menjadi penyebab blue screen  misalnya. Adapun untuk instal ulang graphic adapter langkahnya adalah sbb :

Nyalakan laptop;

Masuk ke safe mode seperti langkah diatas;

Tekan tombol keyboard CTRL + ALT + DELETE untuk memunculkan Task Manager;

Pada jendela taskmanager, tekan CTRL + N atau klik menu File > New kemudian ketik explorer.exe dan tekan Enter;

Setelah Windows Explorer muncul, klik ikon Computer pada panel kiri, dan klik Manage;

Pada jendela Computer management yang muncul, pada panel kiri, klik Device Manager;

Pada panel kanan, expand section Display adapter, klik kanan pada driver yang muncul dan klik Properties;

Pada jendela yang muncul, klik Update driver software, ikuti langkah yang diberikan, hingga selesai;

Pada tahap ini, jika instalasi driver sebelumnya memang memerlukan CD-ROM driver bawaan adapter atau dapat unduh dari internet, maka ada opsi untuk mengambil source instalasi driver dari disk;

Setelah proses selesai, restart laptop, jika masalah driver graphic selesai, seharusny Windows 7 sudah masuk Desktop.

Cara Pertama

Anda coba menekan tombol dengan logo Windows + P, setelah itu tekan tombol yang bergambar panah.

Kemudian lihat dan perhatikan pada windows laptop Anda, apakah kursor di layar keluar, apabila hal tersbut terjadi, tekan dan tahan tombol Spasi dan juga ketik pasword atau sandi, lalu tekan Enter.

Cara Kedua

Lepaskan atau copot Baterai Laptop Anda.

Cabut Casan Laptop Anda.

Tekan dengan menahan tombol Power sampai 10 detik.

Copt tombol Powernya.

Setelah itu pasang semua perangkat yang dilepaskan tadi, kemudian nyalakan kembali.

Cara Ketiga

Pada saat laptop Anda windows nya berlayar hitam atau blackscreen Anda coba dengan cara ini yaitu : Tahan dan tekan tombol ALT + DEL + CTRL dengan bersamaan, yang bertujuan untuk membuka Task Manager, apabila sudah terbuka, Klik File ~ New ~ lalu ketik explorer.exe

Apabila berhasil dengan selesai maka Anda harus melakukan System pada windows untuk di Restore dan aplikasi yang membuat tidak kompatible segera di Uninstal.

Cara Keempat

Yang pertama pada pengaturan windows Background Show harus aktif (Anda bisa lihat di settingan)

Setelah itu masuk ke setting dengan cara menekan tombol logo Window + I atau juga bisa ketik Setting di Menu Start

Kemudian Anda pindahkan jendelanya di Ease of access ~ Other Options. Dengan cara seperti ini yaitu buka Settings — > Lalu Ease of access — > dan Other Options.

Berikutnya harus menyala pada sklar Windows Background Show

Lalu Anda harus memilih thema sesuai keinginan Anda

Yang terakhir Anda harus selesaikan terlebih dahulu kerjaannya, setelah itu keluar dengan ketik Sign out dari laptop windows 10, setelah itu masuk kemabli dnegan Sign In, maka dengan ini layar dekstop sobat akan kembali normal.
