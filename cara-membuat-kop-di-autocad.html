9. Cara membuat kop surat di MW


Pastikan Mode View dalam bentuk Print Layout. Bila belum, anda dapat membuat mode view menjadi print layout dengan cara : pilih menu tab View, pilih tombolprint layout.Setelah dokumen dalam keadaan viewPrint layout, lakukan Double click (klik ganda) pada area header yang terdapat pada bagian atas dari dokumen yang sedang tampil sehingga anda sekarang berada dalam pengeditan HeaderDocument.Bila belum ada, berikan tanda check pada opsi Different First Page. Hal ini untuk membuat Kop surat hanya akan ada di halaman pertama, sedangkan halaman kedua dan seterusnya tidak menggunakan kop surat.Geser Margin kiri anda rada ke tengah. Perkirakan lebarnya kira-kira cukup untuk logo yang akan anda tambahkan belakangan



Ketikan nama lembaga, instansi, atau perusahaan anda pada header tersebut. Tambahkan pula baris kosong pada baris terakhir.Tambahkan pula gambar logo instansi anda. Jangan lupa untuk membuat format layout dari gambar logo tersebut menjadi “In Front of text”.Atur posisi logo dan teks instansi.Pilih dan blok baris kosong yang ada pada baris terakhir untuk dibuat garis pembatas kop surat.



Untuk membuat garis pembatas kop surat, pilih tab menu Home, lalu pilih tombol toolbar border.Dari pilihan popup menu untuk membuat border yang ada, pilih item “Border and shading..” yang ada pada paling bawah



Pada Dialog Border and Shading tersebut pada Setting pilih Custom, kemudian pilih bentuk garis tebal dan tipis, lalu tambahkan pada bagian atas paragraf. Bila anda ingin garis pembatas kop surat berwarna selain hitam, misalkan merah atau biru, silahkan anda ganti saja warnanya. Bila pilihan sudah benar, klik tombol OK.



Kop surat anda hampir siap. Jika border belum rata kiri, anda tinggal menarik margin kiri pada baris terakhir ke arah kiri.



Setelah Margin Kiri sudah digeser ke arah kiri, Kop surat sudah siap digunakan. Klik dua kali pada area dokumen untuk menutup Header View, atau anda dapat langsung klik tab menu Header and footer tools, lalu Design lalu pilih Close Header and Footer.Dokumen anda sudah memiliki Kop surat sudah siap anda gunakan. Jangan khawatir dengan tampilan abu-abu, hal tersebut menunjukkan bahwa Kop surat kita berada di area header dokumen.



10. Jelaskan cara membuat kop surat


Jawaban:

cara membuat kop seurat di microsoft word

1). Buka aplikasi Microsoft Word

2). Atur margin dokumen, dan jenis ukuran kertas yang akan digunakan. Untuk latihan kali ini margin dokumen left: 3, right:2,5, top: 2,5, bottom:2,5 ukuran kertas Legal

3). Klik dua kali pada bagian atas dokumen untuk mengaktifkan fungsi Header. Kemudian beri centang Different First Page fungsinya agar kop surat hanya muncul di halaman pertama saja.

cara membuat kop surat di microsoft word

4). Tulis profil perusahaan / instansi anda, seperti contoh dibawah ini. Atur posisi teks ditengah menggunakan align center.

cara membuat kop surat di microsoft word dengan mudah

5). Masukan Logo Instansi disebelah teks profil instansi, dengan cara klik menu Insert, pilih Picture dan cari lokasi penyimpanan Logo instansi anda. Kemudian setelah logo tampil, agar Logo mudah diatur posisinya, klik kanan pada logo pilih Text Wrapping – pilih In Front of Text.

6). Berikutnya kop surat pasti menggunakan garis (border) dibagian bawah teks dan logo. Langkahnya mudah, letakan kursor dibagian bawah toks dan logo, klik Home, klik icon Border & Shading, klik Border, pilih jenis border pada Style. Dibagian Preview klik border kiri, kanan dan bawah untuk menghilangkan border yang tidak dibutuhkan, selanjutnya klik Ok. Maka dibagian bawah teks sudah tampil border.


11. Cara menggambar rumah di Autocad


Jawaban:

Membuat kapling/denah biasa terlebih dahulu.

Banyak cara yang di gunakan dalam membuat denah rumah ini bisa dengan membuat layernya terlebih dahulu atau dengan command multiline, tapi saya akan menggunakan command yang sudah saya berikan.

command yang di pakai : Line (L), Offset (O)

Gambarlah desain awalnya terlebih dahulu sesuai dengan ukuran yang sudah ada di gambar ini

Untuk ukuran dinding 15 mm

Membuat pintu dan jendelanya di tempat/space yang terpisah dari denah yang di buat tadi untuk lebih gampang di pindahkan nantinya.

command yang di pakai : Rectangle (Rec), Arc, Copy (Co)

Gambarlah sesuai dengan ukuran yang sudah di sediakan seperti gambar di bawah ini

Untuk gambar jendela yang di kotak merah

Memindahkan object pintu yang sudah di buat tadi ke object denah/kapling yang akan di buatkan pintu.

command yang di pakai : Copy (Co), Mirror (Mi), Rotasi (Ro)

Ubah tampilan object pintunya seperti yang di kotak merah dengan menggunakan command Mirror dan Rotasi

Lalu pindahkan object pintu ke space yang dimana akan di jadikan tempat keluar masuk seperti yang di lingkaran merah dengan menggunakan command Copy

sekarang giliran Memindahkan object jendelanya ke space yang menurut kalian cocok untuk di pasang jendela.

command yang di pakai : Copy (Co)

pindahkan object jendela yang sudah di buat sebelumnya (dalam kotak merah) ke space yang di inginkan (dalam lingkaran merah)

anda bisa membuat garis bantu agar mendapatkan midpoint/basepointnya untuk menyesuaikan letaknya

Membuat toilet.

Agar lebih mudah kita membuat toiletnya berada di luar denah, jadi nanti tinggal di pindahkan saja

command yang di pakai : Rec (R), (Offset (O), Arc, Mirror (Mi)

Gambarlah toilet sesuai ukuran seperti gambar di bawah ini dengan menggunakan command2 tersebut atau sesuai keinginan anda juga ga apa2

memindahkan toilet yang sudah di buat tadi ke kamar mandi.

command yang di pakai : Move (M)

pindahkan toilet tadi seperti gambar di bawah ini dengan menggunakan command Move

Membuat bak mandi.

command yang di pakai : Rectangle (Rec)

buatlah bak mandi dengan ukuran yang anda inginkan

membuat teras depan dan belakang dan lantai kamar mandi.

command yang di pakai : Line (L), Hatch (Ha)

membuat areanya terlebih dahulu dengan menggunakan command Line

Lalu memberi tampilan seperti lantai dengan menggunakan command Hatch dengan pengaturan pattern = NET, scale = 300, dan add pick points sesuai gambar di bawah ini

Untuk lantai kamar mandi ukuran scale = 150

Membuat carport/garasi.

command yang di pakai : Line (L), Offset (O), Copy (Co)

Buatlah garasi/carport dengan command2 tersebut seperti gambar di bawah ini atau sesuai keinginan anda

Memberi nama pada masing-masing bagian.

command yang di pakai : Dtext (Dt)

Buatlah nama pada masing-masing bagian dengan menggunakan command Dtext tentunya

Demikian tutorial sederhana membuat gambar rumah dari autocad

13. langkah langkah membuat ukuran garis pada AutoCAD


1. Ketik L enter pada keyboard komputer atau laptop Anda.

2. Kemudian klik pada sembarang display autocad untuk menentukan point pertama.

3. Tekan F8 untuk pembuatan garis lurus searah vertical (Y axis) atau horizontal (X axis)

4. Arahkan mouse ke arah sesuai keinginan, misalnya pada contoh ini saya menggerakan mouse ke arah kanan sehingga sejajar dengan sumbu X.

5. Lalu ketik dimensi atau ukuran panjangnya garis sesuai harapan, seperti terlihat di contoh gambar saya memasukkan dimensi 50.

6. Tekan enter untuk mengakhiri perintah Line di autocad dan lihat hasilnya.
