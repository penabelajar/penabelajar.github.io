4. setiap hari Adel meminum 0,2 liter susu kedelai. ibu memiliki persediaan susu kedelai bubuk sebanyak 3 kotak. setiap kotak dapat digunakan untuk membuat 1,2 liter susu kedelai. Adel akan menghabiskan persediaan susu kedelai bubuk dalam ... hari


Dik :
Adel meminum susu 0,2 liter/hari...
Ibu menyiadakan susu kotak 3 buah, Dan digunakan ibu untuk membuat susu 1,2 liter/kotak..

Dit. Berapa hari adel menghabiskan susu kedelai ?

Jawab...

1 kotak = 1,2 liter susu
3 kotak = 1,2 liter x 3 = 3,6 liter

Adel akan menghabiskan susu :
3,6 liter : 0,2 liter
= 18 hari.

Jadi, Adel akan menghabiskan susu 3 kotak selama 18 hari...
Semoga membantu.. ^_^
1,2×3=3,6÷0,2=18 jadi adel akan menghabiskan susu kedelai dalam jangka waktu 18hari
