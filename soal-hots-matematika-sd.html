= 1/2 + 1/6

Polanya, 
1/(1 x 2) + 1/(2 x 3) + 1/(n x (n + 1))
= n/(n + 1) → di mana harus dimulai dari 1/(1 x 2)

1/(2 x 3) + 1/(3 x 4) + ... + 1/(99 x 100) = n → soal asli
1/(1 x 2) + 1/(2 x 3) + 1/(3 x 4) + ... + 1/(99 x 100) = n + 1/(1 x 2) → ruas kanan dan kiri ditambah 1/(1 x 2)

1/(1 x 2) + 1/(2 x 3) + 1/(3 x 4) + ... + 1/(99 x 100) = n + 1/(1 x 2)
99/100 = n + 1/2
n = 99/100 - 1/2

Maka, 
1/2 - n
= 1/2 - (99/100 - 1/2)
= 1/2 - 99/100 + 1/2
= 1/2 + 1/2 - 99/100
= 1 - 99/100
= 100/100 - 99/100
= 1/100
