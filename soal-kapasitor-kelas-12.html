3. Contoh soal kapasitansi kapasitor


Jawab:

Sebuah kapasitor dipasang pada tegangan 220 volt. Bila muatan yang tersimpan 110 μC, tentukan kapasitas kapasitornya!

Penjelasan dengan langkah-langkah:

Diketahui:

V = 220 volt

Q = 110 μC

Ditanya: C =…?

Pembahasan:

Gunakan persamaan berikut untuk mencari kapasitas kapasitor.

[tex]C=\frac{Q}{V} \\\ \\C=\frac{110}{220} \\\\= 0,5uf[/tex]

Jadi, kapasitas kapasitornya adalah 0,5 μF.

8. tolong dijawab soal fisika tentang kapasitor ini


Pada Rangkaian Capasitansi.

Ketika beberapa capasitor di rangkai secara seri.
1/Cs = 1/C + 1/C +...
1/Cs = n 1/C
Cs = C/n

Ketika n capasitor dirangkai secara paralel.
Cp = C + C +...
Cp = nC

Kemudian rangkaian seri dan paralel di rangkai secara seri

1/C = 1/Cp + 1/Cs

1/C = 1/nC + n/C
1/C = (1/nC + n^2/nC)
1/C = (1+ n^2) / nC

C = nC / (1 + n^2)

Jadi, capasitor penggantinya adalah nC/(1+n^2)

10. Sebuah kapasitor 2µF diberi beda potensial 12 V, hitunglah energi yang tersimpan pada kapasitor tersebut?


◾ Materi : Kapasitor

◾ Mapel : Fisika
◾ Keyword : Energi pada Kapasitor

Diketahui :
c = 2 μF = 2x10^-6 F
v = 12 volt

Ditanya :
W = ?

Penyelesaian :
Energi kapasitor jika diketahui kapasitor dan beda potensial adalah
W = ½cv²
W = ½(2x10^-6)(12)²
W = 144x10^-6 J
W = 1,44 x 10^-4 J

Jadi, energi yg tersimpan pada kapasitor adalah 1,44 x 10^-4 J

11. sebuah kapasitor 12 uf dihubungkan dengan sumber tegangan 24v.sedangkan kapasitor 8 pf dihubungkan dengan tegangan 16 v jika kedua kapasitor digabungkan hitunglah muatan akhir setiap kapasitor


Jawaban:

q₁ = 2,88×10⁻⁴ C

q₂ = 1,92×10⁻¹⁰ C

Penjelasan:

muatan masing-masing kapasitor adalah:

q₁ = C₁.V₁

q₁ = 12μF . 24

q₁ = 288×10⁻⁶ C

q₂ = C₂.V₂

q₂ = 8 pF . 16

q₂ = 128 pC = 128×10⁻¹² C

potensial gabungan kedua kapasitor:

Vgab = (q₁ + q₂) / (C₁ + C₂)

Vgab = (288×10⁻⁶ + 128×10⁻¹²) / (12×10⁻⁶ + 8×10⁻¹²)

Vgab = 2,88×10⁻⁴ / 1,2×10⁻⁵ = 24 volt

setelah digabungkan muatan akhir setiap kapasitor adalah:

q₁ = C₁ . Vgab

q₁ = 12×10⁻⁶ . 24

q₁ = 2,88×10⁻⁴ C

q₂ = C₂ . Vgab

q₂ = 8×10⁻¹² . 24

q₂ = 1,92×10⁻¹⁰ C
