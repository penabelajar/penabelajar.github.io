Nomor 1.
Agar dapat berdampingan, maka diberikan tiga grup warna, dengan ketiganya dapat saling bertukar tempat (3!)
Pada tiap grup, tiap bola dapat ditukar pada masing-masing grupnya:
→ Hitam ada 2 bola (2!)
→ Merah ada 3 bola (3!)
→ Putih ada 5 bola (5!)

Kalikan seluruhnya:
= 3! x 2! x 3! x 5!
= 6 x 2 x 6 x 120
= 8.640 cara


Nomor 2.
Paling sedikit satu gambar dari pelemparan 3 uang logam:
→ Satu gambar, dua angka = {(G,A,A), (A,G,A), (A,A,G)}
→ Dua gambar, satu angka = {(G,G,A), (G,A,G), (A,G,G)}
→ Tiga gambar, tanpa angka = {(G,G,G)}
Banyaknya anggota = n(A) = 7

Dengan ruang sampel untuk tiga mata uang = n(S) = 2³ = 8

Maka, peluangnya adalah:
P = n(A)/n(S)
P = 7/8


Nomor 3.
Karena hanya mengambil satu secara acak, tidak perlu menggunakan kombinasi  untuk mempercepat pengerjaan.
Dengan banyaknya biru = n(Biru) = 4
Dengan banyaknya seluruhnya, n(S) = 4+20 = 24

Maka:
Peluang terambilnya kelereng biru:
P(Biru) = n(Biru)/n(S)
P(Biru) = 4/24
P(Biru) = 1/6
