3. Bagaimana teknik gambar pot bunga


Mengumpulkan apa yang Anda butuhkan. Dapatkan selembar kertas yang bagus, pensil tajam, karet. Jika Anda ingin membuatnya berwarna-warni, Anda bisa mendapatkan beberapa warna juga. Untuk memulai sebuah karya seni, Anda selalu perlu untuk mengumpulkan persediaan Anda sehingga Anda dapat berkonsentrasi pada gambar daripada barang-barang Anda

Membuat struktur dasar mulai dengan sebuah persegi panjang dengan garis yang mencuat dari itu. Hal ini akan bertindak sebagai dasar dari gambar. Selalu ingat, tidak pernah menggunakan penggaris untuk menggambar garis atau bentuk apapun. Jika Anda tidak dapat menarik garis lurus tanpa penggaris, sedikit latihan bisa membantu Anda

Membuat tabel di bawah persegi panjang (yang kemudian menjadi pot). Afterall, pot harus pada sesuatu. Hal ini tidak bisa mengapung di udara. Tapi untuk sekarang, tidak membuat meja yang sangat rinci. membuatnya sederhana. By the way, jangan gunakan penggaris untuk itu. Untuk menggambar meja menggambar bentuk elips dan menarik lain di bawah setengah lebih rendah untuk memberikan tampilan 3D. Gambarlah kaki meja di bawahnya. Anda bisa menggambar pula kaki Anda inginkan. Anda juga bisa membuat sebuah meja persegi panjang jika Anda ingin

Tambahkan daun pada baris mencuat dari persegi panjang. Ini adalah daun yang bisa membuat sesuatu terlihat seperti tanaman. Jangan menarik daun sangat rinci hanya menarik beberapa sekop di sekitar garis

Tambahkan baris lain. Kami tidak ingin batang untuk melihat 1D. Seharusnya terlihat 3D

Sekarang datang merinci bagian. Pertama membuat sisi pot miring. Menghapus bagian atas panci dan menggambar elips. Bergabung dengan sisi pot dengan elipse dan kemudian menggambar elipse yang lebih kecil dalam satu lebih besar. Berikan efek untuk itu agar terlihat seperti tanah dan menarik beberapa garis kecil di tempat di mana tanaman terpasang dengan tanah agar terlihat tidak teratur. Membuat sekop pada tanaman terlihat seperti daun dengan menarik garis kecil di bagian tengah dari masing-masing daun. Buatlah ujung-ujung daun bulat

Sekarang menambahkan rincian ke meja. menarik beberapa garis membulat di bagian atas meja agar terlihat seperti kayu. Tambahkan beberapa garis pada kaki juga. Membuatnya terlihat tidak teratur seperti kayu dan tidak sempurna seperti marmer

Tambahkan beberapa warna jika Anda ingin dan Voila!

10. Kuis [+50]: Di lapangan ditaruhlah pot-pot bunga. Di baris pertama ditaruhlah pot bunga sebanyak 91 pot bunga, baris ketiga ada 85 pot bunga. [a] Di baris berapa terdapat 1 pot bunga saja? [b] Berapakah jumlah semua pot bunga yang ditaruh di lapangan?


Ini adalah Barisan Aritmatika.

Setiap baris berkurang 6 Pot Bunga.

Rumus suku ke n nya :

Un = a + (n-1) x b

U1 = a = 91

U3 = 85

U3 = a + 2b

a + 2b = 85

91 + 2b = 85

2b = -6

b = -3

Maka Baris yang terdapat 1 Pot Bunga :

1 = 91 + (n-1) x (-3)

1 = 91 - 3n + 3

1 = -3n + 94

3n = 93

n = 93/3 = 31

Jawaban A = Baris ke 31

B. Jumlah Pot Bunga = Sampai suku ke 31 atau sisa 1 Pot Bunga.

Sn = n/2 (a + a + (n-1) x b)

S31 = 31/2 (91 + 91 + 30 x (-3))

S31 = 15,5 (182 + (-90))

S31 = 15,5 x 92

S31 = 1426 Pot Bunga Jawabannya.

Di  setiap baris  terdapat 1 pot bunga saja terdapat 31

jumlah semua pot bunga yang ditaruh di lapangan sebanyak 1.426

Diketahui :

Di lapangan ditaruhlah pot-pot bunga. Di baris pertama ditaruhlah pot bunga sebanyak 91 pot bunga, baris ketiga ada 85 pot bunga.

Ditanya :

[a] Di baris berapa terdapat 1 pot bunga saja?

[b] Berapakah jumlah semua pot bunga yang ditaruh di lapangan?

Penyelesaian :

Barisan pertama = 91

Barisan ke 3 = 85

Rumus barisan arimatika :

Sn = n : 2 x { a + a + (n - 1 ) x b }

Un = a + ( n - 1 ) x b

U1 / a

b = selisih setiap pola

Un = a + 2b

U3 = a + 2b

U85 = a + 2b

U85 = 91 + 2b

2b = 85 - 91

2b = - 6

b = - 6 : 2

b = - 3

Massukan b = 3 , ke rumus Un arimatika

Un = a + ( n - 1 ) x b

1 = 91 + ( n - 1 ) x ( - 3)

1 = 91 + ( - 3n + 3 )

1 = 91 - 3n +  3

1 = - 3n + 94

- 3n = - 94 + 1

- 3n = - 93

n = - 93 : ( - 3 )

n = 31

Berapakah jumlah semua pot bunga yang ditaruh di lapangan?

Sn = n : 2 x { a + a + ( n - 1 ) x b }

S31 = 31 : 2 x { 91 + 91 + (  31 - 1 ) x ( - 3 )

S31 = 15.5 { 91 + 91 + 30 x ( - 3 ) }

S31 = 15.5 x { 91 + 91 - 90 }

S31 = 15,5 x 182 - 90

S31 = 15.5 x 92

S31 = 1426

Ibu memiliki 150 pot bunga lalu ibu membeli lagi 6 lusin pot bunga. Lalu pot bunga di sumbangkan ke sekolah sebanyak 50 pot bunga berapa pot bunga ibu sekarang.

『Jawaban』

1 Lusin = 12 buah × 6 = 72 pot

150 + 72 = 222 pot

Disumbangkan 50 pot = 222-50 = 172 pot

⇔Jadi, sisa pot bunga ibu adalah = 172 pot
