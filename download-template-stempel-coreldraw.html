11. Jelaskan kegunaan dari fasilitas New Graphics, Open Last Edited, Template, dan Corel Tutor pada aplikasi CorelDRAW? #thanks


- New Graphic digunakan untuk membuat sebuah file obyek gambar yang baru.
- Open Last Edited untuk membuka file-file CorelDraw yang pernah dibuat dan tersimpan dalam hardisk.
- Template untuk membuat obyek gambar dengan menggunakan fasilitas template(pola-pola/desain) yang telah disediakan oleh pihak Corel atau yang dibuat oleh si user.
- Corel Tutor untuk membuka lembaran tutorial yang disediakan oleh CorelDraw.

Penjelasan:

Misalkan welcome to coreldraw 12, maka akan tampil icon berikut ini:

- New: icon untuk membuat file baru.

- Recently Used: icon yg menampilkan file-file yang telah dipakai sebelumnya.

- Open: icon untuk membuka file yang sebelumnya pernah dibuat

- New From Templates: icon untuk membuat file baru dg template yg tersedia

- CorelTUTOR: icon untuk menampilkan tutorial pemakaian Corel Draw

- What's New: icon untuk menampilkan hal-hal baru dalam Coreldraw
