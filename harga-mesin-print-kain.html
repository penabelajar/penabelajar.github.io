Sofware merupakan bagian dari sistem komputer. Software berbentuk data yang di simpan dan diatur oleh komputer dalam sebuah program untuk menjalankan suatu perintah. Sotware memiliki fungsi sebagai penggerak atau pengontrol komputer agar dapat menjalan sebuah perintah dari penggunanya.  

Software komputer juga berperan sebagai pengidentifikasi program yang dijalankan oleh user. Selain itu, software dalam sebuah komputer juga berperan sebagai pengatur hardware-hardware komputer untuk bekerja secara bersamaan.

Software pada komputer memiiki beberapa jenis, yakni freeware, shareware, firmware, commercial software, free software, open source software, dan malware. Contoh-contoh dari software yang banyak digunakan oleh user komputer adalah.
