1. Sebutkan langkah-langkah membuat data supplier pada aplikasi Myob accounting ​


Jawaban:

1.lik Card File → Card List

2.Klik Customer → New, kemudian masukkan data suplier

3.Klik Buying Detail kemudian lengkapi rincian pembelian termasuk pajak dan

syarat pembayaran (isilah sesuai ketentuan dari masing-masing suplier)

4.Kemudian klik OK

Jawaban:

1.Mencatat Pelanggan & Pemasok

Pelanggan (customer) adalah pihak yang membeli barang dagang kita. Sedangkan pemasok (supplier) adalah pihak yang menjual barang dagang yang kita beli. Karena seringnya terjadi transaksi dengan suplier dan customer maka sebaiknya dibuat suatu database untuk menyimpan customer dan suplier tersebut. Data ini nantinya akan berhubungan dengan Utang (dengan suplier) dan piutang (dengan customer).

2.Langkah pertama adalah mengatur setting File baru, bila belum jelas, anda bisa mempelajari Setting awal Myob pada postingan terdahulu. Setelah anda berada di halaman awal Myob, maka langkah berikutnya adalah Mencatat Data Pelanggan (Customer) dan Data Pemasok (Supplier). Langkahnya adalah sebagai beikut :

3.Klik Card File, kemudian Klik Card List.

4.Setelah itu Pilih bagian Customer atau Supplier

Untuk mengisi data Customer, Pilih bagian Customer, jika yang di isi data Supplier, maka pilih juga Supplier. Cara Pengisian data dari keduanya hampir sama.

Sebagai contoh mengisi Customer, Pastikan yang aktif adalah Bagian Customer dan klik New, maka akan tampil gambar dibawah ini.

5.Kemudian isi

Name : Toko.ABC (nama Customer), kemudian Enter

ID : C01 (kode customer)

Kemudian isi alamat, Kota dan data-data lain jika ada, atau tidak di isi semua juga tidak apa-apa, yang penting Name dan ID yang harus di isi. Kemudian Klik OK.

Lalu klik New kembali untuk mengisi data Customer berikutnya. Silahkan masukkan menurut anda sendiri dan untuk Kode Customer di urutkan saja yaitu C02 dan seterusnya.

Setelah selesai Isi data, misalnya

Name : Toko.Riman (nama Supplier), kemudian Enter

ID : S01 (kode Supplier)

6.Kemudian isi alamat, Kota dan data-data lain jika ada, atau tidak di isi semua juga tidak apa-apa, yang penting Name dan ID yang harus di isi.

9. sebutkan syarat pembayaran yang ditentukan untuk customer dan supplier


Syarat pembayaran itu ada 2 yaitu on cash dan kredit. On cash dilakukan apabila pembayaran dilakukan pada saat penyerahan barang sedangkan kredit di dilakukan apabila pembayaran dilakukan setelah barang diserahkan, kredit ada pula yang dilakukan pembayaran secara bertahap atau angsuran. Dalam pembayaran sistem kredit biasanya ada yang disebut dgn termin pembayaran. Termin pembayaran itu seperti kita tau yaitu EOM (End Of Month) atau 2/10 n/30 atau n/30 dsb.

11. cara memperbaiki data dan penyetelan myob​


Jawaban:

1. Pilih Create, kemudian akan tampil gambar sebagai berikut.

2. Klik next saja, karena itu adalah informasi awal program Myob.

3. Langkah berikutnya adalah isi data perusahaan, termasuk nama Perusahaan, alamat, No. Telepon, Fax dam Email, lalu klik Next.

4. Langkah berikutnya adalah setting periode Akuntansi.

Untuk setting diatas, anda harus pastikan settingan tahun benar yaitu 2008 dan settingan bulan juga harus tepat yaitu di akhiri bulan desemberpada Last month of financial (bulan akhir periode akuntansi) serta bulanJanuari pada Conversion Month (bulan awal periode akuntansi). Karena kalau anda sala, makapada waktu menjurnal nantinya akan berpengaruh. Kemudian klik Next.

5. Maka akan tampil informasi yag telah di set tadi. Pada tahap ini cukup klikNext saja.

6. Pada langkah ini anda diperlihatkan 3 pilihan untuk memilih Perkiraan.

Pilihan pertama = bila menggunakan perkiraan dari program Myob

Pilihan kedua = bila ingin mengimpor perkiraan dari perusahan lain / file lain

Pilihan ketiga = bila ingin membuat sendiri perkiraan tersebut.

Anda boleh memilih apa saja, karena nantinya akan anda set juga perkiraannya, standarnya pilih saja No. 1 lalu klik Next.

7. Kemudian silahkan jenis Perusahaan anda, dan type bisnis dari perusahaan tersebut. Ini pun boleh anda abaikan, lalu klik aja Next.

8. Pada langkah ini tentukan tempat penyimpanan file anda.

Setelah itu, silahkan klik Next. maka tampil proses dari settingan awal file anda.

9. Kemudian pada pilihan terakhir , pilihCommand Center.

15. langkah langkah dalam membuat backup file data myob adalah dengan cara?


Berikut ini adalah langkah-langkah yang bisa kamu lakukan dalam membuat backup file data MYOB:

Aktifkanlah data file "MYOB" lalu klik "File" pada menu baris, kemudian klik "Backup" jendela MYOB maka "Premier MYOB Company File Backup" akan tampil. Pilih "Backup Company File Only", kemudian klik.Pilih "Check Company File" for errors, kemudian klik.Klik tombol Continue, lalu klik tombol OK dan simpan file backup ditempat yang diinginkan dengan meng-klik tombol Save.

Pembahasan

Backup merupakan proses menyalin atau duplikasi data dan file dari satu perangkat atau media penyimpanan ke media sekunder lainnya, seperti hardisk, flashdisk, sistem cloud dan media lainnya baik dilakukan secara offline maupun online.

Tujuan backup ialah untuk mengembalikan data apabila data tersebut hilang, baik karena rusak ataupun terhapus, dan untuk mengembalikan data ke titik tertentu pada masa lalu. Fungsi utama backup ialah membuat cadangan dari data-data agar ketika data tersebut rusak atau hilang kita masih memiliki salinannya.

Sistem backup terdiri dari beberapa komponen, yaitu:

Software backup merupakan perangkat lunak yang bertugas untuk menjalankan proses backup.Client backup merupakan komputer yang berisi data yang akan dibackup.Server backup merupakan server yang bertugas untuk mengatur proses backup.Media backup merupakan media seperti tape, magnetic disc, optical disc dan sebagainya.Mesin backup merupakan mesin yang digunakan untuk membackup data.
