7. 10 jenis file source code


10 jenis file source code.

Dalam pemrograman, source code adalah file dimana kode program ditulis. Setiap bahasa pemrograman dan IDE (Integrated Development Environment) memiliki ekstensi file sendiri. Misalnya:

1. *.cpp : jenis file source code berbahasa C++

2. *.c : jenis file source code berbahasa C

3. *.h : jenis file header source code berbahasa C, merupakan file yang berisi keterangan variabel dari source code c/c++

4. *.cls : jenis file source code dari class, biasanya ditulis menggunakan bahasa C#

5. *.java : jenis file source code bahasa java.

6. *.sln : jenis file source code dari solution yang di generate oleh program MS Visual Studio, berisi konfigurasi project.

7. *.html : jenis file source code halaman web

8. *.php : jenis file source code dari program backend website yang ditulis dengan bahasa php.

9. *.pas : jenis file source code berbahasa pascal

10. *.js : jenis file source code dari kode javascript

14. Jelaskan source code diatas tiap barisnya, hasil runnya, dan bahasa planet mana yang digunakan dalam source code diatas?


wakakak bahasa planet katanya :D itu bahasa C#
baris 1 menampilkan tulisan yang memerintahkan kita untuk menginputkan angka
baris 2 angka pertama yg sudah d inputkan tadi di masukkan kedalam memory sebagai variabel x
baris 3 dan 4 sama, angka ke 2 di simpan sebagai variabel y
baris 5,6,7 x dan y d bandingkan, angka yang lebih besar akan d tampilkan ke layar sebagai variable lebihbesar
baris 8 menampilkan x dan y sebelum d tukar
baris 9 ada sebuah proses untuk menukarkan x dan y
baris ke 10 menampilkan x dan y yang sudah d tukar
baris ke 12 menampilkan perintah untuk menginputkan nilai faktorial
baris ke 13 inputan d baca sebagai variable x
baris 14 sebuah proses faktorial yang d namakan ok, dan muncul hasil sebagai variabel f
baris 15,16 ada perbandingan, jika ok (faktorial jalan) maka tampilkan faktorial dari huruf x sebagai huruf f
