MAKING AN APPLICATION LETTER

 

Jawaban  Pendahuluan  

Application letter adalah surat yang dibuat dengan tujuan untuk melamar pekerjaaan.

Generic Sturcture dari Application Letter antara lain:

1. Salutation : Berisi tentang salam pembuka

2. Introduction : Berisi tentang kalimat pembukaan. Termasuk tujuan menulis surat tersebut serta nama posisi yg dilamar.

3. Body of the letter: berisi tentang data diri pelamar beserta kemampuan-kemampuan yang dimiliki pelamar yang sesuai dengan pekerjaan tersebut.

4. Closing: Berisi harapan-harapan pelamar agar dapat diterima di perusahaan tersebut.

5. Complimentari Close: Berisi salam penutup dan tanda tangan serta nama terang pelamar.


Language Features dari Application Letter:

1. Menggunakan Simple Present Tense

2. Menggunakan simple future untuk memberikan solusi terhadap pekerjaan yang dilamar.

14. jumlah warga lulusan SMA sebanyak empat kali lipat warga lulusan PT. Jika jumlah seluruh warga adalah 120. berapakah persentase warga lulusan SMA?​


Jawab:

Dik : seluruh warga = 120 orang

       SMA = 4x PT = 40

       SD = 25

       SMK = 15

       PT = 10

       SMP = ??

Dit : Presentase lulusan SMA ???

----------------------------------------------------------------------

*) Mencari presentase masing - masing

SMA = [tex]\frac{40}{120} x 100 % = 33,3 %[/tex] = 33,3 %

PT = [tex]\frac{10}{120} x 100[/tex] = 8,33 %

SD = [tex]\frac{25}{120} x 100 = 20,83[/tex] %

SMK = [tex]\frac{15}{120} x 100 = 12,5 %[/tex] %

SMP = 100 % - 33,3 % - 8,33% - 20,83 % - 12,5% = 25.04 %
