Jawab:

turunan

i. y = axⁿ  --> y' = an xⁿ⁻¹
ii.  y  = u/v --> y' = (u'v -  u v') / v²

turunan dari

[tex]\sf f(x) = \dfrac{4x-x^2}{4\sqrt x}[/tex]

u = 4x - x²
u' = 4 - 2x

v = 4√x

[tex]\sf v' = \dfrac{2}{\sqrt x}[/tex]

y' = (u'v -  u v') / v²

[tex]\sf y' = \dfrac{(4-2x)(4\sqrt x) - (4x-x^2)(\frac{2}{\sqrt x})}{(4\sqrt x)^2}[/tex]

[tex]\sf y' = \dfrac{16\sqqrtx- 8x\sqrt x - (8\sqrt x - 2x\sqrt x)}{16x}[/tex]

[tex]\sf y' = \dfrac{16\sqqrtx- 8x\sqrt x - 8\sqrt x + 2x\sqrt x}{16x}[/tex]

[tex]\sf y' = \dfrac{16\sqqrtx- 6x\sqrt x - 8\sqrt x }{16x}[/tex]

[tex]\sf y' = \dfrac{2(8\sqqrtx- 3x\sqrt x - 4\sqrt x) }{2(8x)}[/tex]

[tex]\sf y' = \dfrac{8\sqqrtx- 3x\sqrt x - 4\sqrt x }{8x}[/tex]

   interval cekung ke atas: {x | x > 1}
