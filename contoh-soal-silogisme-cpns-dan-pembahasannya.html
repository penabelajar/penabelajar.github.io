2. Contoh soal silogisme dalam matematila


Silogisme berarti “jika diketahu p → q dan q→r, maka bisa ditarik kesimpulan p→r“. sebagai contoh :
Premis 1 : Jika harga BBM naik, maka harga bahan pokok naik.
Premis 2 : Jika harga bahan pokok naik maka semua orang tidak senang
maka kesimpulannya adalah Jika harga BBM naik, maka semua orang tidak senang.

Jawaban:

Silogisme adalah suatu proses penarikan kesimpulan secara deduktif. Silogisme disusun dari dua proposisi (pernyataan) dan sebuah konklusi (kesimpulan).

6. Contoh kalimat silogisme negatif, dan silogisme error, lengkap dengan premis-premisnya.


Contoh kalimat silogisme negatif beserta premisnya yaitu:

Premis umum (PU): Semua penderita penyakit gula darah tidak boleh makan terlalu banyak gula.

Premis Khusus (PK): Luna menderita penyakit gula darah

Simpulan (S): Luna tidak boleh makan terlalu banyak gula.

Contoh kalimat silogisme error (salah) beserta premisnya yaitu:

PK: Lulu rajin menabung.

PK: Lulu menjadi kaya raya.

S: Orang yang rajin menabung akan kaya raya?

Dari contoh di atas tidak terdapat Premis Umum (PU).

Pembahasan:

Silogisme adalah sebuah proses berfikir yang bersebrangan dari satu atau lebih premis, yaitu, Pernyataan – pernyataan yang mengawalinya agar dapat diambil suatu kesimpulan berdasarkan prinsip – prinsip logis perlawanan ataupun pendasaran yang cukup.

Di dalam silogisme terdapat dua kategori premis dan satu kesimpulan. Kedua premis itu adalah premis umum (premis mayor) dan premis khusus (premis minor).

Premis Umum ( =PU ) menyatakan jika seluruh anggota golongan tertentu (= semua A) mempunyai sifat ataupun hal tertentu ( =B ). Premis Khusus ( =PK ) menyatakan jika sesuatu ataupun seseorang ( =C ) merupakan suatu anggota golongan tertentu itu ( =A ).

Sedangkan simpulan ( =S ) menyatakan jika sesuatu atau seseorang itu ( =C ) mempunyai sifat ataupun hal tersebut pada B ( =B ).

Jenis Jenis Silogisme.

Silogisme Negatif

Silogisme Negatif bisa dilihat dengan adanya kata “bukan atau tidak” pada premis dan begitu juga simpulan. Jika satu premis pada silogisme bersifat negatif, maka simpulannya bersifat negatif juga.

Contoh :

PU : Semua yang masuk sekolah tidak boleh telat.

PK : Rino masuk sekolah.

S : Rino tidak boleh telat.

Silogisme Error (salah)

Dalam sebuah argumentasi kita harus berhati-hati menggunakan penalaran silogisme. Jika tidak berhati-hati maka akan timbul masalah ketika mengambil simpulan. Jika merumuskan premis harus dengan cermat sebab kesalahan sering terjadi pada penyusunan premis. Untuk menghindari kesalahan perlu diperhatikan peringatan dibawah ini :

Jika terdapat dua premis khusus, maka tidak bisa ditarik simpulan yang dapat dipertanggung jawabkan kebenarannya

PK: Anang diterima sebagai mahasiswa UI.

PK: Anang remaja yang cerdas.

S: Remaja yang cerdas diterima sebagai mahasiswa UI?

Dari contoh tersebut di atas tidak terdapat PU.

9. Contoh soal penarikan kesimpulan yg sah dalam modus silogisme


Contoh soal : modus silogisme 1. Tentukan kesimpulan dari premis berikut ini :) Jika x bilangan real, maka x2 ≥ 0 Jika x2 ≥ 0, maka (x2 + 2) > 0 Pembahasan : Untuk mempermudah, lakukan pemisalan sebagai berikut : » x bilangan real = A » x2 ≥ 0 = B » (x2 + 2) > 0 = C Dengan menggunakan kaidah silogisme, maka pernyataan di atas dapat disusun menjadi : A ⇒ B B ⇒ C ∴ A ⇒ C Dengan demikian, kesimpulan dari premis di atas adalah : Jika x bilangan real, maka (x2 + 2) > 0. 2. Diketahui pernyataan sebagai berikut : Jika Rihanna konser di Jakarta, maka saya akan menonton Jika saya menonton, maka saya sangat senang Tentukan kesimpulan yang sah tentang pernyataan di atas. Pembahasan : Rihanna konser di Jakarta : P Saya menonton : Q Saya sangat senang : R Dengan menggunakan silogisme, maka pernyataan di atas dapat disusun menjadi : P ⇒ Q Q ⇒ R ∴ P ⇒ R Jadi, kesimpulan dari pernyataan di atas adalah : Jika Rihanna konser di Jakarta, maka saya sangat senang.
