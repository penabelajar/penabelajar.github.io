2. Soal Matematika SMP Nilai rata rata ulangan matematika kelas A adalah Xa dan kelas B adalah Xb. Setelah kedua kelas digabung, nilai rata ratanya adalah X. Jika Xa : Xb = 10 : 9 dan X : Xb = 85 : 81 maka perbandingan banyak siswa kelas A dan B adalah ..... A. 3 : 5 B. 5 : 4 C 5 : 3 D. 4 : 5


Jawaban:

Xa : Xb = 10: 9 = 90:81

X : Xb = 85 : 81

Xa : Xb : X = 90 : 81 : 85

                   = 90n : 81n : 85n

AXa + BXb / (A +B) = X

A.90n + B.81n = 85n ( A +B)

90A + 81B = 85A + 85B

90A – 85A = 85B – 81B

5A = 4B

 A : B = 4 : 5

Perbandingan banyak siswa A dan B = 4 : 5
