7. 1). Jika A dan B bekerja bersama-sama dapat menyelesaikan pekerjaan selama 20 dari. B dan C bekerja bersama-sama dapat menyelesaikan pekerjaan selama 12 hari, sedangkan A dan C dapat menyelesaikn pekerjaan itu selama 10 hari. Dalam berapa harikah mereka dapat menyelesaikan pekerjaan apabila : 1. A, B dan C bekerja bersama-sama 2. Mereka bekerja sendiri-sendiri 2). Aku adalah bilangan terdiri atas 3 angka. Jumlah angka – angka bilangan penyusunku 20. 2 kali angka ratusan ditambah 3 kali angka satuan hasilnya 31. Jika 5 kali angka puluhan dikurang 4 kali angka satuan hasilnya 15. Tentukan angka- angka bilanganku.


Jawaban:

Lama A dan B bekerja bersama = 20 hari

Lama B dan C bekerja bersama = 12 hari

Lama A dan C bekerja bersam = 10 hari

Ditanya :

Dalam berapa hari mereka dapat menyelesaikan pekerjaan apabila mereka mengerjakan pekerjaan mereka sendiri sendiri ?

Penjelasan dengan langkah-langkah:

Bagian pekerjaan yang bisa diselesaikan dalam 1 menit secara sendiri-sendiri

A = 1/A bagian

B = 1/B bagian

C = 1/C bagian

Kita buat persamaan dari penyataan diatas

1/A + 1/B = 1/20 ... pers I

1/B + 1/C = 1/12 ... pers II

1/A + 1/C = 1/10 ... pers III

Jumlahkan persamaan I, II, dan III

1/A + 1/B = 1/20

1/B + 1/C = 1/12

1/A + 1/C = 1/10

------------------------------- +

2 (1/A) + 2 (1/B) + 2 (1/C) = 1/20 + 1/12 + 1/10

2 (1/A + 1/B + 1/C) = 3/60 + 5/60 + 6/60

2 (1/A + 1/B + 1/C) = 14/60

1/A + 1/B + 1/C = 14/60 × 1/2

1/A + 1/B + 1/C = 7/60 ... pers IV

subtitusikan persamaan IV, jika 1/B + 1/C = 1/12

1/A + 1/B + 1/C = 7/60

1/A + 1/12 = 7/60

1/A = 7/60 - 1/12

1/A = 7/60 - 5/60

1/A = 2/60

A = 60/2

A = 30 hari

subtitusikan pers I, jika A = 30

1/A + 1/B = 1/20

1/30 + 1/B = 1/20

1/B = 1/20 - 1/30

1/B = 3/60 - 2/60

1/B = 1/60

B = 60 hari

subtitusikan pers III, jika A = 30

1/A + 1/C = 1/10

1/30 + 1/C = 1/10

1/C = 1/10 - 1/30

1/C = 6/60 - 2/60

1/C = 4/60

C = 60/4

C = 15 hari

Jadi lama mereka dapat menyelesaikan pekerjaan apabila mereka mengerjakan pekerjaan mereka sendiri sendiri adalah

A = 30 hari

B = 60 hari

C = 15 hari.

8. sebuah bilangan terdiri atas tiga angka. Jumlah angka angka tersebut 12. Tiga kali angka pertama di tambah angka kedua hasilnya 8. Angka pertama dikurangi 2 kali angka kedua, kemudian di tambah angka ketiga hasilnya -3 . Bilangan yang di maksud adalah


dimisalkan saja :
angka pertama    = x
angka kedua       = y
angka ketiga       = z
diketahui :
x + y + z = 12  ....(persamaan 1)
x - 2y + z = -3  ....(persamaan 2)
3x + y = 8  ..........(persamaan 3)
JAWAB.
1. eliminasi y pada persamaan 1 dan 2
    => 3x + 3z = 21  ....(persamaan 4)
    => 3z = 21 - 3x
    => z  = 7 - x
2. substitusikan z=7-x pada persamaan 1
   => x + y + z = 12
   => x + y + 7 - x = 12
   => x - x + y = 12 - 7
   => y = 5
3. substitusikan y = 5 pada persamaan 3
   => 3x + y = 8
   => 3x + 5 = 8
   => 3x = 8 - 5
   => 3x = 3
   => x = 1
4. substitusikan x = 1 dan y = 5 pada persamaan 1
   => x + y + z = 12
   => 1 + 5 + z = 12
   => z = 12 - 1 - 5
   => z = 6
Jadi, ketiga bilangan yang dimaksud adalah 1, 5 dan 6.

11. Diketahui susunan lima angka 1 2 3 4 5. Sebuah tanda tambah (+) dan sebuah tanda kurang (–) disisipkan di antara dua angka yang berdekatan sehingga jika dihitungmenghasilkan suatu bilangan dengan nilai tertentu. Berikut dua contoh penempatan tanda: a) 1 + 234 –5 = 230 b) 12 – 3 + 45 =54 Selisih bilangan terbesar dengan bilangan terkecil yang mungkin dihasilkan dengan cara tersebut adalah….


Kategori Soal : Matematika
Kelas : 6 SD
Materi : Operasi Hitung Campuran
Kata Kunci : susunan, angka, bilangan

Pembahasan :

Dari susunan 1 2 3 4 5, dapat diketahui bahwa bilangan terbesar yang dapat dihasilkan adalah :

1 - 2 + 345 = 344

Lalu, bilangan terkecilnya adalah :

1 + 2 - 345 = -342

Jadi, selisih antara bilangan terbesar dan bilangan terkecil yang mungkin adalah :

344 - (-342) = 686

3) a+b+c deret aritmatika jumlah 12

a+b+(c+2) deret geometri

Karena a,b,c aritmatika, maka berlaku :

a + (a + b) + (a + 2b) = 12

3a + 3b = 12

a + b = 4

b = 4 - a

Karena a,b,c+2 geometri, maka berlaku :

[tex]\frac{a+b}{a}=\frac{a+2b+2}{a+b}\\{(a+b)}^2=a(a+2b+2)\\4^2=a(a+8-2a+2)\\16=10a-a^2\\a^2-10a+16=0\\(a-2)(a-8)=0\\a=2\,atau\,a=8[/tex]

Ambil a = 2, sehingga b = 2. Dengan demikian, diperoleh deret bilangan tersebut yaitu :

2 + 4 + 6

Jadi, hasil kali ketiga bilangan tersebut adalah 2 x 4 x 6 = 48.
