Harga Tas = 200.000

Harga Coklat = 60.000

⏩ Penyelesaian soal:

Diketahui:

Harga tas sekolah: 80.000 ( lebih dari 2 kali harga kotak coklat ) Total harga tas dan sekotak coklat adalah: 260.000

Ditanya:

harga tas sekolah ?

Jawab:

=========

Misal:

Tas = x

Coklat = z

=========

x + z = 260.000

2z + 80.000 + z = 260.000

3z + 80.000 = 260.000

3z = 260.000 - 80.000

3z = 180.000

z = 60.000

Harga Tas:

x = 80.000 + 2z

x = 80.000 + ( 2 × 60.000 )

x = 80.000 + 120.000

x = 200.000

Pembuktian:

x + z = 260.000

200.000 + 60.000 = 260.000

260.000 = 260.000 => Benar ✅
