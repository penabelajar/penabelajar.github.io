5. buat contoh editing dan coding​


Jawaban:

Pemeriksaan (editing)

Pekerjaan pertama pada fase pengolahan data ialah melakukan pemeriksaan (editing) terhadap catatan-catatan hasil observasi, wawancara dan pengisian kuesione

Pengkodean (Coding)

Pengkodean (coding) adalah memberikan kode-kode atau tanda-tanda terhadap catatan–catatan observasi, wawancara dan kuesioner beserta isi/jawaban- nya. Kode itu dapat berupa huruf, angka-angka untuk nomor ataupun untuk nilai, lambang-lambang dan sebagainya. Maksud pengkodean ini adalah untuk mempermudah pengolahan (analisis) data, terutama jika data/informasi itu dianalisis melalui table-tabel (analisis).

14. Contoh dari overloading method , codingan nya dong . mkasih


OVERLOADING
class Pertambahan{
public void tambah1(){
int a=5, b=10;
System.out.println("Hasil Pertambahann dari metod tambah1 ke-1 = "+(a+b));
}
//Metod tambah1 di overloading dengan 2 parameter (int x, int y)
public void tambah1(int x, int y){
System.out.println("Hasil Pertambahann dari metod tambah1 ke-2 = "+(x+y));
}
public static void main(String [] args){
Pertambahan pp;
pp = new Pertambahan();
pp.tambah1();//memanggil metod tambah1 ke-1
pp.tambah1(5,5);//memanggil metod tambah1 ke-2
}
}
