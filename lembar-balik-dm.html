misal harga tiket dewasa=d dan tiket anak2=a
2d+3a=315.000|×1| 2d + 3a=315.000
3d+1a=280.000|×3|9d+3a=840.000
-7d=-525.000
d=75.000
3d+1a=280.000
a=280.000-3(75.000)
a=55.000 jadi harga selembar tiket dewasa dan dua lembar tiket anak-anak=75.000+110.00
=185.000
jadi kembaliannya adalah
200.000-185.000
=15.000
Misalkan :
Harga tiket dewasa = x
Harga tiket anak-anak = y
Maka :
2x + 3y = 315.000 (×1)
3x + y = 280.000 (×3)

2x + 3y = 315.000
9x + 3y = 840.000
----------------------------- -
-7x = -525.000
x = -525.000/-7
x = Rp 75.000,00

2x + 3y = 315.000
2(75.000) + 3y = 315.000
150.000 + 3y = 315.000
3y = 315.000 - 150.000
3y = 165.000
y = 165.000/3
y = Rp 55.000,00

Harga tiket
= x + 2y
= 75.000 + 2(55.000)
= 75.000 + 110.000
= Rp 185.000,00

Uang kembalian
= 2 × 100.000 - 185.000
= 200.000 - 185.000
= Rp 15.000,00

Jadi uang kembalian yang diterima Amir adalah Rp 15.000,00

= 2 (360 + 300 + 270)

= 2 × 930

= 1.860 cm²

L karton = p × l

= 31 × 20

= 620 cm²

maka jumlah karton yang dibutuhkan adalah

1.860 cm² : 620 cm² = 3 lembar
