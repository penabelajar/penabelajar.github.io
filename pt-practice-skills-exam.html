11. Jelaskan hubungan antara technical skills dan managerial skills?


Hubungan antara technical skills dan managerial skills adalah Semakin tinggi tingkatan managerial /technical yang diduduki,  maka semakin tinggi pula managerial skills dan  technical skills yang perlu dimiliki.  technical skills adalah ilmu teknis melakukan sesuatu sedangkan managerial skills adalah ilmu manajemen/leadership dllPembahasan

Ada  tiga tingkatan manajerial: Manajemen Puncak (Top Management), Manajemen  Menengah (Middle Management) dan Manajemen Bawah/Lini

Semakin tinggi tingkatan managerial/technical yang diduduki,  maka semakin tinggi pula managerial skills dan technical skills yang perlu dimiliki.

Technical skills adalah ilmu teknis melakukan sesuatu, keahlian khusus yang harus dimiliki oleh manajer karena ia memiliki tanggung jawab utama yang harus dijalankan. Contoh technical skills adalah cara ngoding untuk membuat aplikasi android, cara analisis data menggunakan python dll.

Sedangkan managerial skills adalah ilmu manajemen/leadership dll seperti berkomunikasi efektif, berorganisasi, bekerja dengan tim dll.

Dialog diatas adalah contoh dialog conditional sentence type 1.

Conditional sentence type 1, menjelaskan pengandaian di masa mendatang yang belum/mungkin terjadi.

Rumusnya adalah:

If + subject + verb 1 + object, subject + will + verb 1 + object

Pada  conditional sentence type 1, kita bisa menggunakan modals sebagai klausa utama selain future tense untuk menyatakan degree of certainty, permission, or a recommendation about the outcome.

Contoh:

If you drop that glass, it will breakIf you don't enter the house now, you will be locked If you want to pass the exam, you should study moreYou can come over if you have time
