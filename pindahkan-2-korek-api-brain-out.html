11. Teka teki korek api 3+1=7 pindahkan satu batang korek api untuk membenarkan perhitungan (setiap garis adalah batang korek api)


Teka teki memindahkan satu batang korek api agar perhitungan 3 + 1 = 7 benar adalah pindahkan 1 batang korek api yang tegak pada tanda + ke bawah angka 7, sehingga:

tanda + menjadi tanda –.Angka 7 menjadi angka 2.Perhitungan menjadi benar yaitu 3 – 1 = 2.

Pembahasan

Batang korek api yang dibutuhkan:

Angka 3 tersusun dari 5 batang korek api.Tanda + tersusun dari 2 batang korek api.Angka 1 tersusun dari 2 batang korek api.Tanda = tersusun dari 2 batang korek api.Angka 7 tersusun dari 3 batang korek api.

Perhitungan dari 3 + 1 = 7 adalah perhitungan yang salah, seharusnya 3 + 1 = 4. Hanya dengan memindahkan 1 batang korek api, ubah perhitungan dari batang korek api tersebut menjadi benar.

Caranya:

1 batang korek api yang tegak kita pindahkan sehingga tanda + menjadi tanda –.1 batang korek api tersebut kita pindahkan ke bawah angka 7 secara mendatar, sehingga angka 7 menjadi angka 2.

Diperolehlah perhitungannya menjadi:

3 – 1 = 2

dan perhitungan ini sudah benar.

Ada satu kemungkinan lagi yaitu 1 batang korek api tersebut kita pindahkan ke tanda sama dengan sehingga tanda = menjadi tanda ≠, dan diperolehlah pernyataan yang benar yaitu:

3 – 1 ≠ 7

Untuk lebih jelasnya dapat dilihat di lampiran.
