2. Apa fungsi Group Create pada MS Word 2010?​


fungsinya untuk membuat tulisan pada amplop surat ataupun label dengan ukuran dan bentuk sesuai dengan pilihan atau keinginan kita.

Group ini hanya terdiri dari 2 (dua) ikon saja yaitu Envelopes dan Labels, berikut selengkapnya.

• Envelopes, berfungsi untuk membuat serta mencetak tulisan tentang pengirim dan penerima surat pada amplop yang akan digunakan untuk mengirim surat.

• Labels, berfungsi untuk membuat serta mencetak tulisan pada label sesuai dengan bentuk dan ukuran label yang akan diberi tulisan.


3. tuliskan menu yang terdapat dalam group milik word 2010​


Jawaban:Tab menu file

Dalam tab menu file berisi ikon-ikon sebagai berikut :

Save : untuk menyimpan dokumen yang telah disimpan setelah diubah.

Save As : untuk meyimpan dokumen baru atau dokumen lama dengan nama yang berbeda.

Open : untuk membuka kembali dokumen yang telah disimpan

Close : untuk menutup lembar kerja

juga terdapat beberapa perintah seperti: nfo, recent, new, print, save and send, help, options, dan close.

Tab menu home

Tab ini paling sering digunakan, karena menggabungkan semua fitur format teks seperti font, dan pengaturan paragraf. Dalam menu home berisi ribbon untuk pengaturan standart, antara lain :

Ribbon Clipboard : berisi ikon untuk mengedit teks.

Ribbon Font : untuk mengatur huruf

Ribbon Paragraph : untuk mengatur paragraph teks

Ribbon Styles : untuk mengatur model penulisan

Ribbon Editing : untuk mencari karakter, kalimat pada dokumen atau menyeleksi teks pada dokumen

3. Tab menu Insert

Tab ini digunakan untuk memasukkan berbagai item dalam dokumen seperti gambar, clip art, halaman cover, tabel, text box, penomeran halaman, dan lain-lain.

Ribbon Pages : untuk penambahan halaman

Ribbon Tables : untuk menyisipkan tabel

Ribbon Illustrations : untuk menyisipkan berbagai gambar

Ribbon Links : untuk menghubungkan antarhalaman atau dengan file lain

Ribbon Header & Footer : untuk membuat catatan dokumen dan penomeran halaman

Ribbon Text : untuk menyisipkan variasi teks pada dokumen

Ribbon Symbols : untuk menyisipkan simbol

4. Tab menu page layout

Tab ini digunakan untuk mengatur halaman seperti margin, tema, orientasi, dan lain-lain.

Ribbon Themes : untuk mengatur tema dokumen

Ribbon Page Setup : untuk mengatur halaman dokumen seperti batas, posisi, ukuran kertas pada dokumen

Ribbon Page Background : untuk mengatur warna latar halaman dokumen

Ribbon Paragraph : untuk mengatur paragraf

Ribbon Arrange : untuk mengatur letak gambar dan teks

5. Tab menu References

Tab ini menyediakan banyak solusi sederhana untuk menghasilkan dokumen

Ribbon Table of Contents : untuk membuat daftar isi secara otomatis

Ribbon Footnotes : untuk menyisipkan catatan kaki

Ribbon Citations and Bibliography : memasukkan kutipan dan daftar pustaka dalam dokumen yang sedang aktif

Ribbon Captions : untuk menambahkan keterangan seputar gambar

Ribbon Index : untuk mengatur indeks pada teks

Ribbon Table of Authority : untuk mengatur kutipan dalam teks

6. Tab menu Mailings

Tab ini digunakan untuk mengirim surat seperti mencetak amplop dan label

Ribbon Create

Ribbon Start Mail Merge

Ribbon Write & Insert Fields

Ribbon Preview Results

Ribbon Finish

7. Tab menu Review

Tab ini untuk membuat perubahan dalam dokumen yang dimungkinkan karena masalah ejaan atau tata bahasanya

Ribbon Proofing

Ribbon Language

Ribbon Comments

Ribbon Tracking

Ribbon Changes

Ribbon Compare

Ribbon Protect

8. Tab menu View

Tab ini digunakan untuk mengubah tampilan dokumen kita ke dua halaman yang berbeda dan memperbesar/memperkecil halaman

Ribbon Document Views

Ribbon Show

Ribbon Zoom

Ribbon Windows

Ribbon Macros

5. Group page setup dan themes pada Ms. Word 2010 terdapat pada menu.​


Jawaban:

Icon Perintah di Group Themes

1. Themes: digunakan untuk memilih tema halaman yang meliputi warna halaman berikut warna hurufnya.

2. Colors: digunakan untuk mengatur warna tema halaman.

3. Fonts: digunakan untuk mengatur tema huruf yang akan diterapkan ke halaman dokumen aktif.

4. Effect: digunakan untuk mengatur tema efek terhadap objek Shape yang terdapat di halaman dokumen.

-Icon Perintah di Group Page Setup

1. Margins: digunakan untuk mengatur batas teks di halaman dokumen.

2. Orientation: digunakan untuk mengatur posisi kertas.

3. Size: digunakan untuk mengatur ukuran halaman.

4. Columns: digunakan untuk mengatur jumlah kolom teks.

5. Breaks: digunakan untuk mengatur kontinuitas halaman maupun kolom teks.

6. Line Numbers: digunakan untuk mengatur kontinuitas nomor baris teks.

7. Hypenation: digunakan untuk memisahkan suku kata dengan tanda hubung secara otomatis.

-Icon Perintah di Group Page Background

1. Watermark : digunakan untuk memberikan efek tanda air di belakang teks.

2. Page Color : digunakan untuk mengatur warna latar halaman.

3. Page Borders : digunakan untuk mengatur garis tepi halaman.

-Icon Perintah di Group Paragraph

1. Left: menggeser tulisan yang dibog di lembar kerja Microsoft Word dari kiri ke kanan.

2. Right: menggeser tulisan pada lembar kerja dari kanan ke kiri halaman.

2. Before: membuat jarak antara dari baris pertama dan baris ke dua dan seterusnya.

3. After sama dengan fungsi dari Before.

-Icon Perintah di Group Arrange

1. Position: meletakan gambar pada lembar kerja sesuai dengan kebutuhan.

2. Warp Text: menyisipkan gambar kedalam naskah yang ada pada pada lembar kerja Microsoft Word.

3. Bring Forward: mentukan picture ke arah depan.

4. Send Backward: mentukan picture ke arah belakang pada lembar kerja.

5. Selection Pane: memilih jenis gambar yang ingin dimasukan pada lembar kerja.

6. Align: membuat tulisan pada lembar kerja pada Microsoft Word untuk rata kanan dan rata kiri.

7. Group: membuat sekelompok group menjadi 1 kelompok, sesui dengan kebutuhan.

8. Rotate: mengatur gambar dengan cara memutar gambar desngan susuai kebutuhan

14. Sebutkan dan jelaskan toolbar yang terdapat pada group font pada Ms. Word 2010!​


Jawaban:

Grup Clipboard terdiri dari beberapa ikon, yaitu:

Tempel , kerjakan untuk mengatur hasil dari Copy atau Cut.

Potong , jalankan untuk mengunduh sekaligus meng-Copy (memindah) teks / objek terpilih.

Salin , lakukan untuk menggandakan atau memperbanyak teks / objek terpilih.

Format Painter , bekerja untuk menirukan atau memberikan bentuk teks format / objek yang sama ke dalam teks / objek lain.

Font Grup ini terdiri dari ikon-ikon, sebagai berikut:

Font , lakukan untuk memilih jenis huruf yang kita inginkan dalam teks.

Ukuran Fons , bekerja untuk menentukan ukuran huruf kecilnya besar (teks).

Grow Font , bekerja untuk memperbesar ukuran huruf dengan cepat.

Kecilkan Font , kerjakan untuk memperkecil ukuran huruf dengan cepat.

Ubah Kasus , sedang dilakukan untuk status huruf kapital atau huruf kecil.

Clear Formating , siap untuk membersihkan pemformatan teks terpilih.

Berani , bekerja menebalkan teks dipilih.

Cetak miring , dilakukan untuk memiringkan / membuat miring teks terpilih.

Garis bawahi , bekerja untuk memberi garis bawah pada teks terpilih

Aneh , bekerja untuk memberikan coretan pada teks terpilih.

Subscript , bekerja untuk membuat karakter mirip perpangkatan tetapi di bagian bawah huruf / angka normal.

Superscript , mengerjakan untuk membuat karakter perpangkatan (normal di bagian atas huruf / angka normal).

Efek Tex , bekerja untuk memberikan efek artistik pada teks terpilih.

Warna Sorot Teks , berfungsi untuk memberikan warna pada lembar atau latar belakang teks terpilih (seperti stabilo).

Warna font , siap untuk disetujui.

Peluru , bekerja untuk memberikan format peluru pada setiap paragraf yang dipilih.

Penomoran , bekerja untuk memberikan format penomoran pada setiap paragraf terpilih.

Daftar Multilevel , bekerja untuk memilih bentuk paragraf berjenjang.

Kurangi Indent , lakukan untuk menggeser ke kiri (kurangi indentasi) pada paragraf terpilih.

Tingkatkan Indent , bekerja untuk menggeser ke kanan (menambah indentasi) pada paragraf terpilih.

Sortir , kerjakan untuk mengurutkan data berdasarkan Alfabet melalui urutan angka.

Tampilkan Tanda Paragraf , memudahkan untuk memperbaiki / memperbaiki tanda koreksi paragraf (mempermudah melihat batas-batas teks).

Align Text Left , berfungsi untuk membuat teks dengan paragraf rata kiri.

Pusat , bekerja untuk teks dengan paragraf rata tengah.

Align Text Righ , mengerjakan untuk teks dengan paragraf rata kanan.

Keadilan , bekerja untuk teks dengan paragraf rata kanan dan kiri.

Penjarangan Baris , dilakukan untuk jarak antar baris teks.

Shading , bekerja untuk membuat warna latar belakang pada teks terpilih.

Perbatasan , bekerja untuk memberikan garis tepi pada teks terpilih.

Dalam tampilan defauld (baku) grup Editing ini terdiri dari:

Temukan , lakukan untuk mencari kata tertentu pada suatu file / dokumen berdasarkan kata kunci (kata kunci) yang dimasukkan.

Ganti , lakukan untuk mencari dan ganti kata yang ditemukan dalam file / dokumen dengan kata tertentu.

Pilih , lakukan untuk memilih objek dan atau teks tertentu dalam file / dokumen.
