5. Sebutkan Fungsi dari masing - masing isi item pada Menu Bar File (Info, New, Open, Save, Save As, Print, Share, Export, Close, Account, Option) ?


1.. Info.

Memunculkan tab informasi dokumen meliputi ukuran, judul, tag, kategori dokumen (Properties), tanggal pembuatan, dan pembaruan (Related Dates), penulis (Related People), dan mengunci memberikan pasword untuk melindungi privasi dokumen (Protect Workbook)

2.New.

Membuat dokumen baru, mencari dan menerapkan template atau tema online ke dalam project.

3.Open.

Membuka dokumen atau berkas yang sudah pernah dibuat, entah itu dalam komputer (This PC), atau One Drive.

4.Save.

Pada menu file microsoft excel, ikon atau perintah save memiliki fungsi untuk menyimpan dokumen ke dalam komputer maupun OneDrive. Shortcutnya: ctrl & S.

5.. Save As.

Perintah save as pada menu file microsoft excel ber-fungsi untuk menyimpan dokumen yang sudah ada dengan nama yang berbeda. Misalkan: Formulir 1 (Sudah ada), menjadi Formulir 2 (Nama lain yang sudah diisi). Sehingga terdapat 2 berkas yang berbeda. hal ini biasanya digunakan untuk sebagai backup antara file awal, dan yang sudah diedit, atau diperbarui.

6.Print.

Group Print pada menu file microsoft excel ber-fungsi untuk mengatur dokumen sebelum dicetak, meliputi jenis printer, halaman, margins, ukuran kertas, dan jumlah salinan. Akan muncul tab preview di samping kanan untuk mengetahui tampilan dokumen jika dicetak.

7.Share.

Share adalah ikon dalam menu File microsoft excel yang memilki fungsi membagikan dokumen ke orang lain melalui email, maupun OneDrive. Terdapat 2 opsi sharing anatar lain:

share pada menu file excel befungsi

Share with People.

Membagikan project yang dibuat ke orang lain yang terhubung dalam lingkup OneDrive.

Email.

Mengirimkan salinan dokumen melalui surat email, dalam bentuk lampiran.

Send a Link.

Menyispkan sebuah link yang berisi dokumen excel, untuk kebutuhan pengerjaan dokumen dengan banyak orang

8.Export.

Fungsi dari Export pada menu file microsoft excel adalah untuk mengubah format/type dokumen ke format lain seperti txt. Ada 2 sub bagian export antara lain:

export pada menu file

Create PDF/XPS Document.

Untuk membuat file menjadi tipe PDF/XPS.

Change File Type.

Mengubah lembar kerja ke dalam tipe dokumen lain seperti excel 97-2003, Spreadsheet, Txt, CVS dll dengan Save maupun Save As.

9.Close.

Fungsi utama Close yang merupakan bagian dari menu file adalah menutup dokumen yang sedang dikerjakan, tanpa mengakhir software microsoft excel.

10.Account.

Tab account yang dapat diakses melalui menu file excel, ber-fungsi untuk mengelola pengaturan akun, tema sistem, login office, dan informasi dari product office, seperti update terakhir, dan informasi software yang digunakan.

11.Options.

Fitur Options adalah sebuah fitur yang terdapat pada menu file excel, yang terdiri atas: General, Formulas, Data, Proofing, Save, Language, Ease of Acces, Advanced, Customize Ribbon, Quick Acces Toolbar, Add-ins, Trust Center, yang memilliki fungsi untuk mengatur toolbar seperti menyembunyikan, dan memunculkan Ribbon.

8. bagaimana cara mengexport gambar pada CorelDraw?​


Langkah 1

Buka file CorelDRAW dan aktifkan halaman yang berisi objek yang perlu Anda ekspor.

Langkah 2

Pilih Ekspor dari menu File. (File> Ekspor) Ini akan memunculkan kotak dialog ekspor.

Langkah 3

Pilih jenis format file yang diinginkan yang ingin Anda ekspor sebagai. (JPG, TIFF, BMP, EPS, dll.) Berikan path dan nama untuk file yang diekspor.

Langkah 4

a) Untuk mengekspor seluruh halaman - Pastikan ‘Hanya yang dipilih’ tidak dicentang.

b) Untuk mengekspor hanya sebagian desain atau beberapa objek dari halaman - Pastikan untuk memilih objek yang diinginkan dari desain dan centang kotak centang ‘Dipilih saja’

Langkah 5

Klik tombol Ekspor. Ini akan memunculkan kotak dialog. Isi dalam kotak dialog tergantung pada jenis file yang Anda pilih untuk ekspor. Masukkan informasi sesuai kebutuhan Anda di kotak dialog. Biasanya, pengaturan dan nilai-nilai 'Terakhir digunakan' dipertahankan oleh DRAW.

Langkah 6

Klik Oke.

13. Cara membuat logo menggunakan coreldraw X6


Awla mula create new. Atur tinggi dan lebar 500 px * 500 px. Lalu klik OK.


Langkah kedua buat text. Dengan cara teka F8 pada keyboard atau tools yang ada pada pojok kiri layar anda. Disini saya membuat text NS dan NUSANTARA. Pertama yang akan kita convert yaitu text NS.

Langkah ketiga. Pada text NS akan kita convert to curves. Pertama blok text tersebut lalu arrange>convert to curves atau CTRL+Q.

Langkah keempat yaitu setelah di convert maka kita akan menambahkan add perspektif. Pertama klik effect>add perspektif. Kemudian atur sedemikan rupa menjadi seperti ini atau mmenurut selera kalian.


Langkah keempat kita akan mendesain tulisan tersebut. Klik effect>extrude. Pada pojok kanan maka akan ada tools extrude. Lalu pilih edit. Small back. Kemudian OK. Setelah itu tekan CTRL+K drag objek text tersebut sampai terpisah dari objek awal.


Langkah keenam berikan warna pada objek tersebut. Tekan F11.

Langkah ketujuh pada text NUSANTARA. Kita curve to convers. Caranya sama seperti diatas.


  


Langkah ketujuh . kita transformasi text. Effect transform->invers. Drag text tersebut hingga pisah dari text asli. Lalu klik shape too pada pojok kiri layar anda. Tarik text tersebut sesuka anda.

Sehingga hasil akhir menjadi seperti ini.


Maksud dari logo tersebut yaitu. NUSANTARA adalah nama Negara Kesatuan Indonesia dari Marauke sampai Sabang. Disini saya lambangkan dengan NS. Pada lamabang tersebut terdapat sebuah gradasi warna biru dan hijau. Warna tersebut saya artikan sebagai kepaluan yang dikelilingi oleh lautan. Karena nusantara kita terdiri dari kurang lebih 17 ribu pulau kecil dan besar. Dimana setiap pulau dikelilingi oleh hamparan lautan nan luas. Sedangkan kata NUSANTARA dengan corak seperti gambar diatas adalah ketidak sengajaan pembuat,karena dilihat menarik. Sebenarnya kata tersebut pasti sudah banyak yang menggunakan tetapi saya jamin tidak ada kata seperti  yang terdapat pada logo saya. Saya berniat memasaangkan logo tersebut pada rancangan web yang akan saya buat. yaitu sebuah situs yang mendeskripsikan tentang NUSANTARA. Dimana pada situs tersebut akan dideskripsikan berapa jumlah penduduk pada pulau A atau Pulau B. serta kota apa saja yang terdapat pada pulua A atau pulau B.
