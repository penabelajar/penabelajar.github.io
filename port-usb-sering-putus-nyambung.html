6. apa fungsi dari USB port?


Fungsi Port USB (Universal Serial Bus) adalah standar bus serial untuk perangkat penghubung, biasanya kepada komputer namun juga digunakan di peralatan lainnya seperti konsol permainan, ponsel dan PDA.
Port USB juga dapat digunakan dalam pengiriman file dari komputer ke android.

7. jelaskan fungsi dari "port serial" dan "USB port"


Fungsi port serial digunakan untuk mengoreksi piranti,seperti mouse, printer, modem dan juga sbg pembaca barcode
Port USB untuk menghubungkan berbagai piranti keras seperti scanner, camera digitl, printer
