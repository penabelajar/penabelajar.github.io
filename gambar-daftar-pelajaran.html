Jawaban:

C. Toko R

Penjelasan:

Toko P

Sepatu = 300.000 x 20/100

= 60.000

Harga sepatu = 300.000 - 60.000 = 240.000

Sandal = 200.000 x 20/100

= 40.000

Harga sandal = 200.000 - 40.000 = 160.000

Harga Total = 240.000 + 160.000 = 400.000

Toko Q

Sepatu = 300.000 x 25/100

= 75.000

Harga sepatu = 300.000 - 75.000 = 225.000

Sandal = 200.000 x 15/100

= 30.000

Harga sandal = 200.000 - 30.000 = 170.000

Harga Total = 225.000 + 170.000 = 395.000

Toko R

Sepatu = 300.000 x 30/100

= 90.000

Harga sepatu = 300.000 - 90.000 = 210.000

Sandal = 200.000 x 10/100

= 20.000

Harga sandal = 200.000 - 20.000 = 180.000

Harga Total = 210.000 + 180.000 = 390.000

Toko S

Sepatu = 300.000 x 15/100

= 45.000

Harga sepatu = 300.000 - 45.000 = 255.000

Sandal = 200.000 x 25/100

= 50.000

Harga sandal = 200.000 - 50.000 = 150.000

Harga Total = 255.000 + 150.000 = 405.000

Jadi Adit harus membeli di toko R karena harganya lebih murah yakni Rp. 390.000

R < Q < P < S
