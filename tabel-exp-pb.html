Sibudi bengkat membawa 150exp, terjual di pagi hari 87 exp, di sore hari 34 exp, di pakek buat alas tidur siang 1 exp, dikasihin ke si jono temennya 5 exp. Sisa nya di return. Bantulah si budi menghitung keuntungannya..


Jawaban:

diketahui : harga beli koran Rp1.000 per exp.

harga jual pagi : Rp2.000 per exp

harga jual sore : Rp1.250 per exp

koran yang dibeli budi 150 exp, terjual 87 exp di pagi, terjual 34 exp di sore, 1 exp untuk alas tidur, dan di berikan ke jono 5 exp, sisa nya return

ditanya : hitung keuntungan penjualan

Jawab : koran yang di return: 150 - 87 - 34 - 1 - 5 = 23 exp

koran non retur : 87 + 34 + 1 + 5 = 127 exp

biaya beli koran non retur : 127 x 1.000 = 127.000

bruto koran jual pagi : 87 x 2.000 = 174.000

bruto koran jual sore : 34 x 1.250 = 42.500

untuk alas tidur dan dikasih jono = 6 x 1.000 = 6.000

maka keuntungan yang diperoleh = (174.000+42.500) - 127.000 - 6.000 = 83.500

tentukanlah turunan dari fungsi dibawah ini : [tex]y = exp( \frac{1}{ {x}^{2} } ) + \frac{1}{exp( {x}^{2}) } [/tex]

Ditanyakan

Turunan dari fungsi pertama ?

Di Jawab

[tex] - 2( \frac{exp( \frac{1}{x {}^{2} }) }{x {}^{3} } + \frac{x}{exp(x {}^{2}) } [/tex]

Pembahasan

[tex]y = exp( \frac{1}{x {}^{2} } ) + \frac{1}{exp(x {}^{2}) } \\ \\ = \frac{1}{e \: x {}^{2} } + \frac{1}{e \: x {}^{2} } \\ \\ y {}^{1} = \frac{1}{e \: x {}^{2} } \times \frac{ - 2}{x {}^{3} } + \frac{o.e \: x {}^{2} - 1.e \: x {}^{2} .2x }{(e \: x {}^{2}) {}^{2} } \\ \\ = \frac{ - 2.e \frac{1}{x {}^{2} } }{x {}^{3} } + \frac{ - 2x.e \: x {}^{2} }{e2x {}^{2} } \\ \\ \\ = - 2( \frac{e \frac{1}{x {}^{2} } }{x {}^{3} } + \frac{x}{ex {}^{2} } ) \\ \\ = - 2( \frac{exp( \frac{1}{x {}^{2} }) }{x {}^{3} } + \frac{x}{exp(x {}^{2} )} [/tex]

Kesimpulan

Jadi, Turunan dari fungsi pertama tersebut adalah
