6. Berapakah Nilai dari sec (1140)°


~tRigonometri

sec (1140) = sec (4.360 - 300)

sec (1140) = sec 300

sec (1140) = sec 60

sec (1140) = 1 / sec 60

sec (1140) = 1 / (1/2)

sec (1140) = 2

Jawab:

Penjelasan dengan langkah-langkah:

[tex]\displaystyle \sec(1140^o) = \sec(1080^o + 60^o)\\\\\sec(1140^o) = \sec(3\cdot360^o + 60^o)\\\\\sec(1140^o) = \sec(60^o) = \frac{1}{\cos(60^o)} = 2\\\\[/tex]

14. Berapakah nila tani dari 1140°?


Jawab :

[tex]\sqrt{3}[/tex]

Penjelasan :

tangen memiliki periode 180° untuk kembali ke titik semula (0);

tan(180°) = 0 dan tan(keliatan 180°) = 0

tan(1140°) = tan(1080° + 60°)

tan(1140°) = tan(6*180° + 60°)

tan(1140°) = tan(0+60°)

tan(1140°) = tan(60°)

tan(1140°) = [tex]\sqrt{3}[/tex]
