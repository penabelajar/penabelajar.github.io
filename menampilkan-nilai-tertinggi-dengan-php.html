2. Tuliskan syntax menampilkan informasi php


Jawaban:

Dengan file

buat file cekphp.php dan tulis sintax berikut kedalam:

<?php  

phpinfo();

kemudian masukkan diserver dan buka dari browser, maka informasi php akan muncul

Dengan command line

buka command line dan ketik

php --version

dan informasi php akan keluar


3. bagaimana cara mengimplementasikan PHP dengan CSS untuk menampilkan data dari database ?


Pertama ada perbedaan antara PHP dengan CSS, PHP adalah bahasa pemrograman sedangkan CSS adalah pelengkap dalam pemrograman web, yang pada artinya CSS bukan lah bahasa pemrograman itu sendiri.

PHP degan CSS, artinya PHP di gunakan dalam CSS. yang seharusnya CSS dengan PHP, yang dengan nalar bahasa indonesia yag baik dan benar dapat di artikan bahwa CSS di gunakan untuk PHP.

penggunaan pengaksesan database menggunakan credential HANYA pada PHP, dengan pengelogikaan sesuai logika masing-masing.
