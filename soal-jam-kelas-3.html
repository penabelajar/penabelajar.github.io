i) 2x - 4 = 3 ========> x= [tex]3\frac{1}{2}[/tex]

kenapa harus positif-negatif ? karena di persamaan mutlak bilangan negatif akan diubah ke bentuk positifnya sehingga 3 = -3

Jawab:

Penjelasan dengan langkah-langkah:

soal 1

|2x- 4| = 3

2x - 4 =  3 atau 2x - 4 = - 3

2x = 3 + 4  atau 2x  =  - 3 + 4

2x = 7  atau 2x = 1

x= 7/2  atau x = 1/2

HP x= { 1/2 ,  7/2 }

soal 2

|2x +  5| ≤ 6

-  6 ≤ 2x + 5 ≤ 6

-6 - 5 ≤  2x ≤ 6 - 5

-11  ≤  2x  ≤  1

-11/2 ≤ x  ≤ 1

HP x = { x | -11/2 ≤ x  ≤ 1 }

soal 3

|1 - 3x| > 2

1- 3x > 2  atau 1 - 3x < - 2

-3x > 2 - 1  atau  -3x < - 2 - 1

-3x > 1  atau -3x < -3

x < - 1/3  atau  x > 1

HP x = { x |   x < -1/3  atau x > 1}

12. Kelas: Kerjakan soal ber 1. 90 km/jam = ..... m/detik 2. 180 m/menit = .... dam/detik 3. 2 m/detik = .... Km/jam​


Jawaban:

1. 90 km/jam = ..... m/detik

1 km = 1000 m

90 km = 90 x 1000 = 90.000 m/jam

1 jam = 3600 detik

Jadi 3600 x 90.000 = 324.000.000 m/detik

2. 180 m/menit = .... dam/detik

1 m = 0,1 dam

180 m = 0,1 x 180 = 18 dam/menit

1 menit = 60 detik

Jadi 18 x 60 = 1080 dam/detik

3. 2 m/detik = .... Km/jam

1 m = 0,001 km

1 m = 0,001 x 2 = 0,002 km/detik

1 jam = 3600 detik

Jadi 0,002 x 3600 = 7,2 km/jam
