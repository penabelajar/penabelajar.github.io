Maka akan terbentuk persegi panjang sebesar ukuran halaman.

15. sebutkan format asli dari corel draw


1. AI - Adobe Illustrator
Format AI atau Adobe Ilustrator adalah format untuk software AI. Kalau anda terbiasa dengan Adobe Photoshop, Adobe Illustrator termasuk keluarganya, namun cara kerja AI kurang lebih sama dengan CorelDraw karena berbasis Vector.
2. PFB - Adobe Type Font
PFB adalah format untuk spesifikasi font yang dikembangkan oleh Adobe Systems untuk typesetting digital profesional, yang menggunakan format file PostScript untuk mengkodekan informasi font.
3. BMP - Window Bitmap
BMP format file, juga dikenal sebagai file bitmap image atau perangkat bitmap independen (DIB) format file atau hanya bitmap, adalah format raster grafis file gambar yang digunakan untuk menyimpan gambar digital bitmap, secara independen dari perangkat display (seperti adapter grafis ), pada Microsoft Windows.
4. BMP - OS/2 Bitmap
BMP format file, juga dikenal sebagai file bitmap image atau perangkat bitmap independen (DIB) format file atau hanya bitmap, adalah format raster grafis file gambar yang digunakan untuk menyimpan gambar digital bitmap, secara independen dari perangkat display (seperti adapter grafis ), pada  sistem operasi OS/2.

5. CAL - CALS Compressed Bitmap
CAL adalah standar untuk pertukaran data grafis. Format ini dikembangkan oleh Amerika Serikat Departemen Pertahanan (DoD) sebagai bagian dari Akuisisi berkelanjutan dan inisiatif siklus hidup Support (CALS). Ini mendefinisikan penyimpanan raster standar data gambar, baik terkompresi atau dikompres menggunakan CCITT Grup 4 kompresi.
6. CGM - Computer Graphics Metafile
CGM atau Computer Graphics Metafile adalah format file standar internasional yang bebas dan terbuka untuk 2D vektor grafis, grafis raster, dan teks, dan didefinisikan oleh ISO / IEC 8632
7. CMX - Corel Presentation Exchange
CMX adalah ekstensi file untuk file grafis yang digunakan dengan perangkat lunak Corel. CMX singkatan Corel Metafile eXchange. file CMX berisi grafis vektor serta metadata yang menggambarkan gambar. file CMS yang digunakan untuk menyimpan Corel Clip Art.
8. CMX - Corel Presentation Exchange Legacy
CMX legacy merupakan bagian dari format CMX, hanya saja format ini merupakan pengembangan dari format  sebelumnya.
9. CPT - Corel PHOTO-PAINT Image
Format file CPT adalah format file grafis yang digunakan oleh beberapa versi dari Corel Photo Paint. 
10. CPT - Corel PHOTO-PAINT 7/8 Image
CPT 7/8 Image merupakan format terbaru untuk menyimpan data ke program Corel PHOTO-PAINT.
11. CUR -  Windows 3.x/NT Cursor Resource
Format CUR -  Windows 3.x/NT Cursor Resource yang digunakan untuk membuat ikon untuk Windows 3.1, Windows NT, dan Windows 95. Mendukung kursor elemen grafis yang digunakan pada Windows pointer. Anda dapat memilih warna untuk masker transparan dan Inverse. CUR - Windows 3.x/NT Cursor Resource hanya mendukung ukuran gambar maksimum 32 × 32 piksel.12. DOC - MS Word for Windows 6/7
Format DOC atau Document adalah format yang digunakan dokumen microsoft word.
13. DWG - AutoCAD
DWG (dari gambar) adalah format file biner proprietary yang digunakan untuk menyimpan dua dan tiga dimensi data desain dan metadata. Ini adalah format asli untuk beberapa paket CAD termasuk DraftSight, AutoCAD, IntelliCAD (dan variannya), Caddie dan Open Desain Alliance aplikasi compliant.
14. DXF - AutoCAD
AutoCAD DXF (Drawing Interchange Format, atau Drawing Efek Format) adalah format file data CAD yang dikembangkan oleh Autodesk, untuk memungkinkan interoperabilitas data antara AutoCAD dan program lainnya.
15. EMF - Enhanced Windows Metafile
EMF adalah ekstensi file untuk metafile yang telah ditingkatkan, format file spool digunakan dalam pencetakan oleh sistem operasi Windows. Ketika pekerjaan cetak dikirim ke printer, jika sudah mencetak file lain, komputer membaca file baru dan menyimpannya, biasanya pada hard disk atau di memori, untuk mencetak di lain waktu.
16. EPS - Encapsulated Postscript
EPS adalah ekstensi file untuk format file grafis yang digunakan dalam gambar berbasis vektor dalam Adobe Illustrator. EPS singkatan Encapsulated PostScript. Sebuah file EPS dapat berisi teks serta grafis.
17. FMV - Frame Vector Metafile
Frame Vector Metafile mengandung grafis vektor, yang terdiri dari poin dan kurva , digunakan untuk menyimpan ilustrasi dan gambar.
18. FPX - Kodak FlashPix Image
FPX adalah format file bitmap di mana foto disimpan lebih dari satu resolusi.
19. GEM - GEM File
GEM adalah format metafile digunakan oleh Ventura Publisher. file GEM berisi informasi gambar vektor dibuat ketika menggunakan program seperti ChemDraw. GEM File umumnya terkait dengan program desktop publishing yang dibuat oleh Ventura.
20. GIF - CompuServe Bitmap
Format gambar bitmap yang dapat bergerak. GIF diperkenalkan oleh CompuServe pada tahun 1987 dan  penggunaannya  meluas di World Wide Web karena dukungan dan portabilitasnya.
