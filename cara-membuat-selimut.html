6. cara rumus mencari SELIMUT kerucut (bukan yang luas selimut)


Jawaban:

Rumus-rumus dalam bangun kerucut

1) Luas alas = π x r²

2) Keliling alas = 2 x π x r atau π x d

3) Luas selimut = π x r x s

4) Luas permukaan kerucut = luas alas + luas selimut

= π x r² + π x r x s

= π x r x (r + s)

5) Volume kerucut = ¹/₃ x π x r² x t

Dengan keterangan

r = jari" alas

d = diameter alas

s = garis pelukis

t = tinggi
