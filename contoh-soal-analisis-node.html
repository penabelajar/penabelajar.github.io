14. Aturan yang harus dipenuhi pada binary search tree adalah a. Nilai node anak harus lebih kecil dari nilai node parent b. Nilai node parent harus lebih kecil dari nilai node anak c. Nilai node kiri lebih kecil dari node parent d. Nilai node kanan lebih kecil dari node parent e. Node root memiliki nilai paling kecil


Jawaban:

c. Nilai node kiri lebih kecil dari node parent

Penjelasan :

Binary Search Tree adalah tree yang terurut (ordered Binary Tree). Aturan yang

harus dipenuhi untuk membangun sebuah BST adalah sebagai berikut:

Semua data dibagian kiri sub-tree dari node t selalu lebih kecil dari data dalam node t itu sendiri.Semua data dibagian kanan sub-tree dari node t selalu lebih besar atau sama dengan data dalam node t.
