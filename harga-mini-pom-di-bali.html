2. Sebuah mobil mengangkut BBM membawa 10.000 liter bensin, yang akan didistribusikan kepada 4 mini pom dengan pembagian sebagai berikut 1/4 bagian untuk pom mini A, untuk pom mini B : 3m³ dan untuk mini pom C mendapat 2500 liter, setelah mengisi pom D ternyata masih ada sisa 500.000 cm Berapa liter bensin yg digunakan untuk mengisi mini pom D?


Jawaban:

1.500 liter

Penjelasan dengan langkah-langkah:

**) 1 dm³ = 1 liter

A = 10.000/4 = 2.500 liter

B = 3 m³ = 3 × 10³ = 3.000 dm³

C = 2.500 liter

500.000 cm³ = 500.000/10³ = 500 dm³

500 = 10.000 - A - B - C - D

500 = 10.000 - 2.500 - 3.000 - 2.500 - D

500 = 2.000 - D

D = 2.000 - 500

D = 1.500 liter

Jawaban:

Diketahui:

volume bensin mula-mula = 10.000 liter

Untuk pom mini A = 1/4 bagian

Untuk pom mini B = 3m³

Untuk pom mini C = 2500 liter

sisa = 500.000 cm³

Diranya: volume bensin utk mini pom D

Penyelesaian:

1 liter = 1 dm³ = 1.000 cm³

1 m³ = 1.000 liter

Jumlah bensin yg diberikan kepada mini pom D

= 10.000 liter - A - B - C - sisa

= 10.000 liter - (1/4 x 10.000 liter) - 3m³ - 2.500 liter - 500.000 cm³

= 10.000 liter - 2.500 liter - 3.000 liter - 2.500 liter - 500 liter

= 1.500 liter

A.6.997 B.6.972 C.4.470 D.1.500


Jawab:

Untuk diketahui : 1 liter = 1 dm³

Mobil membawa BBM 10.000 liter

Untuk mini pom A ¼ bagian

= ¼ x 10.000 liter = 2.500 liter

Untuk mini pom B 3 m³

= (3 x 1.000) dm³ = 3.000 liter

Untuk mini pom C 2.500 liter

Sisa BBM di mobil = 500.000 cm³ = (500.000 : 1.000) dm³ = 500 liter

Maka, mini pom D mebdapatkan BBM sebanyak :

10.000 - 2.500 - 3.000 - 2.500 - 500 = 1.500 liter (D)

11. sebuah pom mini mempunyai persediaan solar 0,24 meter kubik.kemudiaan mendapatkan kiriman dari pertamina 150.000 ml dan terjual 32.000 cc.persediaan solar di pom mini sekarang sama dengan............liter


0,24 m3 + 150.000 ml - 32.000 cc

Bentuk soal pada cerita tersebut adalah selerti di atas :

Karena ditanya lada satuan l,maka semua satuan diubah ke liter :

Catatan :

1 dm3 = 1 liter
1 ml = 0,001 liter
1 cc = 0,001 liter

Maka :

0,24 m3 = 0,24 × 1.000
=> 240 dm3
=> 240 liter

150.000 ml = 150.000 × 0,001
=> 150 liter

32.000 cx = 32.000 × 0,001
=> 32 liter

Jadi :

240 liter + 150 liter - 32 liter
390 liter - 32 liter
358 liter

jadi,sekarang pom mini tersebut memiliki 358 liter.
Persediaan = 0,24 M3 = 240 Liter

Mendapatkan kiriman = 150.000 Ml = 150 Liter

Terjual = 32.000 Cc = 32 Liter

= 240 Liter + 150 Liter - 32 Liter
= 390 Liter - 32 Liter
= 358 Liter

- Jadi,persediaan solar di pom mini sekarang = 358 Liter
