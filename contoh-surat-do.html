45. Change these sentences into Passive Voice!

a. You can do it very well

Jawabannya => It can be done by you very well

Alasannya => Ini adalah bagian modal dimana O + can + be + v3 + by + S

b. I gave you the plan

Jawabannya => You were given the plan by me.

Alasannya => The plan bukan termasuk dalam objek karena dia ada didepan You.Jadi You telah menjadi sasaran untuk diubah menjadi passive.
