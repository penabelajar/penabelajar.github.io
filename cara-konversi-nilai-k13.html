⑥ Jawaban : B
[tex] | \frac{1}{1 + \sqrt{3} } | \: = \frac{1}{1 + \sqrt{3} } [/tex]
= [1(1-√3)] / [(1+√3)(1-√3)]
= (1-√3) / (1-3)
[tex]= \frac{1 - \sqrt{3} }{ - 2} \\ \\ = \frac{1}{2} ( \sqrt{3} - 1)[/tex]

⑦ Jawaban : C
[tex] | \frac{4}{ \sqrt{7} + \sqrt{11} } | = \frac{4}{ \sqrt{7} + \sqrt{11} } [/tex]

= [4(√7-√11)] / [(√7+√11)(√7-√11)]
= [4(√7-√11)] / [7-11]
= [4(√7-√11)] / -4
[tex]= - ( \sqrt{7} - \sqrt{11} ) \\ \\ = \sqrt{11} - \sqrt{7} [/tex]

3. Bagaimana cara menghitung nilai rata2 raport SMP K13 semester 1 sampai 5 tapi hanya 5 mapel??


Penjelasan dengan langkah-langkah:

1. Hitung jumlah nilai 5 mapel di setiap semester

(jika ada nilai "pengetahuan" dan "keterampilan" maka lihat nilai yang pengetahuannya)

2. Misal smst 1 400, smst 2, dst

3. Jumlahkan nilai yang sudah di hitung tadi

smst1+smst2+...+smst5

4. Hasil penjumlahan tadi, dibagi dengan 25.
